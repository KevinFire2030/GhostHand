# 1페이지 사업기획 요약

## 사업명
**메타버스 기반 에이전트 교육 플랫폼**

## 한 줄 정의
사용자가 놀이공원을 탐험하듯 학습하고, 레고블럭 조립하듯 AI 에이전트를 만들며, 미션 완료 시 라이센스와 보상을 받는 게이미피케이션 기반 온라인 교육 플랫폼.

## 문제 인식
기존 온라인 교육은 영상 시청과 문서 중심으로 이루어져 몰입감이 낮고, 특히 AI/에이전트 같은 기술 교육은 초보자에게 어렵고 추상적으로 느껴진다. 학습자는 실제로 만들어보는 경험 없이 개념만 접하고 끝나는 경우가 많으며, 이로 인해 완주율과 학습 만족도가 떨어진다.

## 해결 방안
이 서비스는 교육 과정을 메타버스형 공간으로 재구성하여 사용자가 학습 콘텐츠를 “강의”가 아니라 “탐험”처럼 경험하도록 만든다. 각 존은 에이전트 개념, 아키텍처, 활용 사례, 실습 공간 등으로 구성되며, 핵심 실습 공간인 `에이전트 스쿨`에서는 툴, 스킬, 에이전트를 단계적으로 조립할 수 있다.

즉,
- 메타버스 = 학습 공간
- 레고블럭 조립식 빌더 = 실습 도구
- 게이미피케이션 = 몰입과 완주를 높이는 장치

## 핵심 가치 제안
- 어렵고 추상적인 AI 에이전트 교육을 쉽고 직관적으로 바꾼다.
- 비개발자도 접근 가능한 체험형 학습 구조를 제공한다.
- 학습자가 직접 “만들어본 경험”을 얻도록 한다.
- 라이센스, 배지, 레벨업 요소를 통해 재미와 성취를 제공한다.

## 핵심 기능
1. **메타버스형 학습 공간**
   - 놀이공원처럼 구성된 학습 맵
   - 주제별 학습 존 탐험

2. **에이전트 스쿨**
   - 툴 → 스킬 → 에이전트 단계형 실습
   - 직접 조립하며 배우는 구조

3. **레고블럭 조립식 에이전트 빌더**
   - 자연어 기반 툴 생성
   - 노드/블럭 기반 GUI 조립
   - 스킬 조합 및 에이전트 완성

4. **게이미피케이션 시스템**
   - 미션, 배지, 라이센스, 레벨업
   - 성취 기반 동기 부여

## 타겟 고객
- AI와 자동화에 관심 있는 일반 사용자
- 모바일 AI 에이전트와 OpenClaw에 관심 있는 실무자
- 기업의 AX 교육 담당자
- 청소년/체험형 AI 교육 시장

## 차별점
기존 AI 교육이 “설명 중심”이라면, 이 서비스는 “탐험 + 조립 + 보상” 중심이다. 단순한 강의 시청이 아니라, 사용자가 하나의 공간을 이동하며 직접 만들고 완성하는 경험을 제공한다는 점에서 차별성이 있다.

## MVP 범위
- 메인 로비
- 3~4개의 학습 존
- 기본 에이전트 스쿨
- 툴/스킬/에이전트 생성 흐름
- 라이센스 1종 발급 기능
- 진행 상태 저장

## 기대 효과
- 학습 몰입도 향상
- 완주율 상승
- AI 에이전트에 대한 진입장벽 완화
- 교육 서비스와 에이전트 빌더 제품의 결합 가능성 확보
- 향후 기업 교육, 청소년 교육, 플랫폼 서비스로 확장 가능

## 한 줄 결론
**이 사업은 AI 에이전트 학습을 강의가 아닌 놀이와 조립의 경험으로 바꾸는, 메타버스형 차세대 교육 플랫폼이다.**
