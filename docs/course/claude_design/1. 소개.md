# 소개

## 5~15분, 소개

이번 순서에서는 Claude Design이 무엇인지, 어디에 활용하면 좋은지, 그리고 기존 방식과 비교했을 때 어떤 장점이 있는지 중심으로 소개합니다.

### 1. Claude Design이 무엇인지
Claude Design은 생성형 AI를 활용해 아이디어를 빠르게 구체화하고, 기획과 디자인 초안을 효율적으로 만들어볼 수 있도록 돕는 도구입니다.
단순히 텍스트를 생성하는 수준을 넘어서, 사용자의 의도를 바탕으로 구조를 정리하고 방향을 제안하며 결과물을 발전시키는 데 강점이 있습니다.

쉽게 말하면,
- 막연한 아이디어를 초안으로 바꿔주고
- 초안을 더 구체적인 형태로 발전시키며
- 반복 수정 과정을 빠르게 도와주는
AI 기반 디자인/기획 보조 도구라고 볼 수 있습니다.

### 2. 어디에 활용하면 좋은지
Claude Design은 다음과 같은 상황에서 특히 유용합니다.

- 서비스 기획 초안 작성
- 화면 구성 아이디어 정리
- 기능 설명 문서 초안 작성
- 회의 전 아이디어 스케치
- 협업용 방향성 정리
- 빠른 프로토타입 사고 정리

특히 처음부터 완벽하게 만들기 어려운 작업에서 강합니다.
예를 들어, 머릿속에는 방향이 있는데 문서로 정리되지 않거나,
디자인이나 기획의 출발점을 빠르게 만들어야 할 때 생산성을 높여줄 수 있습니다.

### 3. 접속 방법
Claude Design의 공식 시작 경로는 현재 Claude Help Center의 시작 가이드와 Anthropic 공식 안내를 기준으로 확인할 수 있습니다.
실제 서비스 진입은 일반적으로 Claude 웹 서비스인 `https://claude.com`에서 로그인한 뒤, Claude Design 권한이 열려 있는 계정으로 접근하는 방식입니다.
참고용 공식 문서는 아래 경로입니다.

- 시작 가이드: `https://support.claude.com/en/articles/14604416-get-started-with-claude-design`
- 공식 소개: `https://www.anthropic.com/news/claude-design-anthropic-labs`
- 일반 Claude 접속: `https://claude.com`

로그인 방법은 기존 Claude 계정 로그인과 동일한 흐름으로 이해하면 됩니다.
즉, Claude에 로그인한 뒤 Claude Design이 활성화된 플랜 또는 조직 계정이라면 사용 가능한 형태입니다.

처음 접속했을 때는 다음 흐름으로 설명하면 좋습니다.
- 왼쪽에는 Claude와 대화하는 채팅 영역이 있고,
- 오른쪽에는 결과물이 생성되는 캔버스 영역이 있습니다.
- 사용자는 채팅창에 원하는 결과를 설명하면,
- Claude가 우측 캔버스에 디자인, 프로토타입, 슬라이드, 문서 초안을 생성합니다.

처음 시작하는 가장 쉬운 방법도 구체적으로 안내할 수 있습니다.
1. Claude에 로그인합니다.
2. Claude Design이 열려 있는 계정/조직인지 확인합니다.
3. 새 프로젝트를 만듭니다.
4. 첫 프롬프트에 만들고 싶은 산출물을 구체적으로 입력합니다.
   - 예: "신규 서비스 소개용 랜딩페이지 초안을 만들어줘"
   - 예: "사내 보고용 5장짜리 발표자료 구조를 만들어줘"
5. 생성된 결과를 본 뒤, 채팅 또는 캔버스 인라인 코멘트로 수정 요청을 합니다.

공식 가이드 기준으로는 프로젝트에 문서나 이미지 같은 참고 자료를 붙여서 맥락을 추가할 수 있고,
채팅으로 큰 방향을 바꾸거나, 인라인 코멘트로 특정 요소만 수정하는 식으로 반복 개선하는 것이 기본 사용 흐름입니다.

### 4. 사용료
사용료는 실제로 "Claude 전체 요금제"와 "Claude Design 전용 사용량 정책"을 함께 봐야 정확합니다.
조사 결과, Claude Design은 일반 Claude 기능과 완전히 같은 방식이 아니라 별도 사용량 기준으로 관리됩니다.

확인된 핵심 내용은 아래와 같습니다.

- Claude Design은 Claude의 나머지 기능과 별도로 가격 및 사용량이 측정됩니다.
- Claude Design에는 자체 사용량 추적과 자체 허용량이 있습니다.
- 구독형 플랜에서는 주간 한도(weekly allowance)가 별도로 리셋됩니다.
- 기본 한도를 넘기면 extra usage를 구매해 추가 사용할 수 있습니다.

현재 공식 안내상 Claude Design은 연구 프리뷰 기준으로 아래 구독자에게 제공됩니다.
- Claude Pro
- Claude Max
- Claude Team
- Claude Enterprise

즉, 완전한 무료 플랜 사용자가 바로 Claude Design을 쓰는 구조로 보기보다는,
현재는 유료 또는 조직용 구독 플랜 중심으로 제공된다고 설명하는 편이 안전합니다.

일반 Claude 요금제 참고 예시는 공식 가격 페이지 기준으로 아래와 같습니다.
- Pro: 월 $20, 연간 결제 시 월 환산 $17
- Max: 월 $100부터
- Team: 좌석당 월 $25, 연간 결제 시 좌석당 월 $20
- Enterprise: 별도 문의

다만 강의에서는 "Claude 기본 요금제 가격"과 "Claude Design 사용량 정책"은 구분해서 설명하는 것이 중요합니다.
예를 들어,
- Pro/Max/Team/Enterprise 가입 여부
- Claude Design 사용 가능 여부
- 주간 허용량이 얼마인지
- 한도 초과 시 extra usage 구매가 가능한지
- 조직 계정에서는 관리자가 별도 설정 또는 비용 관리를 하는지
를 함께 확인해야 합니다.

특히 Team과 seat-based Enterprise는 사용자별 주간 허용량이 제공되고,
usage-based Enterprise는 기존 계약의 표준 API 요율로 청구된다고 안내되어 있습니다.
또한 Enterprise 사용자는 초기 시작 단계에서 사용자당 약 20회 정도의 일반적 프롬프트를 커버하는 1회성 크레딧이 제공된다고 명시되어 있습니다.

즉, 사용료 파트의 핵심은
"기능만 보는 것이 아니라, Claude 요금제와 Claude Design 전용 사용량 정책을 같이 확인해야 실제 비용을 정확히 이해할 수 있다"는 점입니다.

### 5. 기존 방식 대비 장점
기존 방식은 보통 사람이 처음부터 구조를 잡고, 내용을 정리하고, 표현을 다듬으면서 초안을 만드는 데 시간이 많이 들어갑니다.
반면 Claude Design을 활용하면 아래와 같은 장점이 있습니다.

- 시작 속도가 빠릅니다.
- 아이디어를 바로 형태로 바꿔볼 수 있습니다.
- 수정 요청을 통해 방향을 빠르게 조정할 수 있습니다.
- 여러 대안을 짧은 시간 안에 비교해볼 수 있습니다.
- 기획과 디자인 커뮤니케이션의 출발점을 쉽게 만들 수 있습니다.

즉, Claude Design의 핵심 장점은
"처음 시작하는 부담을 줄이고, 반복 개선 속도를 높여준다"는 점입니다.

### 6. 이 파트에서 전달할 핵심 메시지
- Claude Design은 초안 작성과 아이디어 구체화에 강한 도구입니다.
- 특히 기획, 디자인, 문서화의 시작 단계에서 활용 가치가 높습니다.
- 접속 방법과 사용료도 함께 이해해야 실제 활용이 쉬워집니다.
- 기존 수작업 대비 더 빠르게 시작하고, 더 빠르게 개선할 수 있습니다.
