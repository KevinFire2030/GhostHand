# Claude Design 60분 특강 계획

## 1. 강의 개요
- 강의명: Claude Design 60분 특강
- 목표: Claude Design의 핵심 개념과 사용 흐름을 빠르게 이해하고, 바로 실무/개인 업무에 적용할 수 있도록 돕기
- 대상: Claude Design을 처음 접하는 사용자, 생성형 AI 기반 디자인/기획 협업 흐름이 궁금한 실무자
- 진행 시간: 총 60분

## 2. 60분 강의 계획

### 0~5분, 오프닝
- 강사 소개
- 오늘 강의 목표 안내
- 강의 흐름 설명

### 5~15분, 소개
- Claude Design이 무엇인지
- 어디에 활용하면 좋은지
- 기존 방식 대비 장점

#### 포함 내용
- 접속 방법
- 사용료

### 15~30분, 기본 사용 방법
- 첫 화면 구성 이해
- 프롬프트 입력 방식
- 결과 확인 및 수정 흐름
- 반복적으로 원하는 결과를 얻는 방법

### 30~45분, 활용 예시
- 기획안 초안 만들기
- 화면/기능 아이디어 확장하기
- 문서 구조 잡기
- 협업 시나리오에서 활용하기

### 45~60분, 실습
- 간단한 주제로 직접 입력해보기
- 결과물을 바탕으로 수정 요청해보기
- 원하는 방향으로 개선하는 반복 실습
- Q&A 및 마무리

## 3. 세부 구성안

### 소개
- Claude Design 개념 소개
- 접속 방법
- 사용료

### 기본 사용 방법
- 시작 화면 이해
- 입력 방법
- 결과 확인
- 수정/재요청 방법

### 활용 예시
- 아이디어 발상
- 기획 문서 작성
- 디자인 방향 정리
- 협업 초안 생성

### 실습
- 실습 주제 안내
- 직접 프롬프트 작성
- 결과 개선 실습
- 질의응답
