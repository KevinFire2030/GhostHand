# Lecture 01. 내 스마트폰에서 돌아가는 AI 비서

## 세션 개요
첫 강의에서는 모바일 AI 에이전트의 개념과 필요성을 소개하고, OpenClaw 기반 구조를 직관적으로 이해시키는 데 집중한다.

## 학습 목표
- 왜 모바일 AI 에이전트인가를 설명할 수 있다.
- OpenClaw 기반 아키텍처를 큰 흐름으로 이해한다.
- 책과 강의의 전체 구조를 파악한다.

## 진행 흐름
### 1. 문제 제기
- 왜 지금 에이전트인가
- 왜 스마트폰인가
- 기존 AI 앱과 무엇이 다른가

### 2. 핵심 개념
- 에이전트
- 메모리
- 툴
- 스킬
- RAG

### 3. 플랫폼 구조 소개
- 사용자 요청
- Gateway / Router
- Agent Runtime
- Memory / Skills / Tools
- Android / External Services

### 4. 데모 예시
- 간단한 요청 수행
- 외부 서비스 연동
- 스마트폰 자동화 사례

### 5. 정리
- 오늘의 핵심 메시지
- 다음 강의 예고

- 실제 사례를 먼저 보여주고 개념 설명으로 연결
