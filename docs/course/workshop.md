# 실습형 워크숍 구성안

## 워크숍 목표
참가자가 OpenClaw 기반 모바일 에이전트의 작동 원리를 이해하고, 최소 1개의 자동화 성공 경험을 직접 만들어보도록 한다.

## 권장 시간
- 3시간 버전
- 4시간 버전

## 구성
### Part 1. 오리엔테이션
- 오늘 만들 것
- 필요한 환경
- 실습 시 주의사항

### Part 2. 환경 준비
- Termux
- OpenClaw
- 기본 연결 확인

### Part 3. 첫 성공 경험
- 간단한 요청 실행
- 응답 확인
- 에이전트 동작 흐름 설명

### Part 4. 자동화 실습
- 단순 반복 작업 자동화
- 외부 서비스 연결 예시
- 개인 시나리오 적용

### Part 5. 마무리
- 실습 회고
- 자주 막히는 포인트 정리
- 다음 단계 제안

## 준비물
- 안드로이드 스마트폰
- 네트워크 환경
- 실습 계정
- 기본 문서/가이드

## 운영 메모
- 환경 이슈 대응 시간을 넉넉히 잡기
- 첫 성공 경험을 가능한 빨리 제공하기
- 참가자별 수준 차이를 고려한 보조 시나리오 준비
