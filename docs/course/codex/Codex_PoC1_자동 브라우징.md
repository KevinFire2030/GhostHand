# Codex PoC 1. 자동 브라우징

## 한 줄 설명

코덱스가 인터넷에서 최신 정보를 직접 찾아보고, 요약하고, 비교하고, 출처까지 붙여서 정리해주는 흐름을 짧게 체험하는 데모입니다.

---

## 이 실습에서 보여줄 것

- 코덱스가 "검색"만 하는 게 아니라, 최신 정보를 직접 확인한다
- 단순 요약이 아니라 비교, 정리, 추천까지 이어질 수 있다
- 링크와 출처를 포함해 결과를 정리할 수 있다
- "최신", "오늘", "지금", "최근" 같은 요청에서 특히 강하다

---

## 진행 멘트 예시

지금부터는 코덱스의 자동 브라우징을 보여드립니다.  
아까까지는 파일 생성, 코드 작성, 문서화 같은 로컬 작업을 해봤다면, 이번에는 인터넷에서 최신 정보를 직접 확인하는 흐름을 실습합니다.  
참가자는 아래 프롬프트를 그대로 복붙해서 붙여 넣으면 됩니다.

---

## 실습 1. 오늘의 AI 뉴스 3개 찾기

### 설명

가장 직관적인 자동 브라우징 데모입니다.  
코덱스가 최신 뉴스를 직접 찾아보고, 핵심만 짧게 정리하는 흐름을 보여줄 수 있습니다.

### 복붙 프롬프트

```text
오늘 AI 관련 주요 뉴스 3개만 찾아서 날짜와 함께 3줄씩 요약해줘. 출처 링크도 같이 붙여줘.
```

### 기대 포인트

- "오늘" 같은 최신성 있는 요청을 직접 확인한다
- 검색 결과를 사람이 읽기 쉽게 요약한다
- 링크까지 붙여서 바로 검증 가능하다

---

## 실습 2. 두 서비스 비교해보기

### 설명

브라우징은 단순 검색보다 비교에서 더 강하게 체감됩니다.  
예를 들어 최신 요금제, 기능, 차이점을 바로 비교하게 해볼 수 있습니다.

### 복붙 프롬프트

```text
ChatGPT와 Claude의 최신 유료 요금제를 찾아서 가격, 주요 기능, 누가 쓰기 좋은지 표처럼 비교해줘. 출처 링크도 넣어줘.
```

### 기대 포인트

- 여러 페이지를 보고 비교 정리할 수 있다
- 참가자가 "내가 지금 선택하려면?" 관점으로 받아들이기 쉽다
- 실무적으로도 바로 쓸 만한 결과물이 나온다

---

## 실습 3. 여행/맛집/구매형 검색 보여주기

### 설명

현실적인 질문일수록 자동 브라우징의 체감이 좋습니다.  
특히 돈이나 시간을 쓰는 의사결정에서 유용하다는 점을 보여주기 좋습니다.

### 복붙 프롬프트

```text
서울 성수동에서 평점 좋고 요즘 많이 가는 브런치 카페 5곳 찾아줘. 왜 추천하는지도 한 줄씩 적어주고 링크도 붙여줘.
```

### 기대 포인트

- 추천형 검색도 가능하다는 점을 보여준다
- 최신 후기, 현재 운영 여부 같은 정보를 반영할 수 있다
- 참가자들이 "이건 바로 써먹겠다"는 반응을 많이 보인다

---

## 실습 4. 특정 페이지 읽고 요약시키기

### 설명

검색만이 아니라, 특정 문서나 페이지를 읽고 정리시키는 데모입니다.  
링크 하나를 주고 핵심만 뽑아달라고 하면 체감이 큽니다.

### 복붙 프롬프트

```text
이 페이지 읽고 핵심만 5줄로 요약해줘. 초보자가 알아야 할 점도 따로 3개 적어줘: https://platform.openai.com/docs/overview
```

### 기대 포인트

- 내가 지정한 문서를 읽고 정리할 수 있다
- 긴 문서를 짧은 학습 자료로 바꿀 수 있다
- 교육, 리서치, 문서 읽기 업무에 바로 연결된다

---

## 실습 5. 최신 정보 + 추천까지 받기

### 설명

검색한 정보를 바탕으로 바로 의사결정까지 연결하는 예시입니다.  
단순 나열이 아니라 추천 이유까지 정리하게 해보면 좋습니다.

### 복붙 프롬프트

```text
2026년 4월 기준으로 입문자가 쓰기 좋은 AI 코딩 도구 5개를 찾아서 장단점과 추천 대상을 정리해줘. 최신 정보 기준으로 링크도 함께 줘.
```

### 기대 포인트

- 검색 이후 해석과 추천까지 이어진다
- "최신 기준" 요청을 자연스럽게 수행한다
- 참가자가 코덱스를 조사 도구로 인식하게 된다

---

## 강의용 추천 진행 순서

1. 뉴스 검색으로 "최신 확인"을 보여준다
2. 서비스 비교로 "정리 능력"을 보여준다
3. 맛집/구매형 질문으로 "생활형 활용"을 보여준다
4. 특정 문서 요약으로 "업무 활용"을 보여준다
5. 추천형 질문으로 "판단 보조"를 보여준다

---

## 발표할 때 같이 말하면 좋은 포인트

- "코덱스는 최신 정보가 필요한 질문에서 직접 확인을 시도합니다."
- "검색만 해주는 게 아니라, 읽고 정리하고 비교해줍니다."
- "링크를 같이 주기 때문에 검증하면서 쓸 수 있습니다."
- "특히 최신 뉴스, 가격, 문서, 제품 비교, 추천형 질문에서 강합니다."

---

## 한 줄 마무리

자동 브라우징 데모는 "최신 정보 확인 -> 요약 -> 비교 -> 추천" 흐름으로 보여주면 가장 직관적이고 반응이 좋습니다.
