# 처음 만나는 코덱스

## 코덱스 처음 실행하고 해보는 10가지 초간단 실습

1시간 특강에서 화면에 띄워두고 그대로 따라 하기 좋게 정리한 문서입니다.  
실습자는 `나` 블록만 그대로 복붙해서 입력하면 되고, `코덱스` 블록은 어떤 식으로 진행되는지 보여주는 예시입니다.

---

## 시작 전에

- 코덱스는 질문만 하는 도구가 아니라, 파일 생성, 코드 작성, 수정, 문서화, Git 작업까지 같이 할 수 있습니다.
- 아래 실습은 아주 작은 성공 경험을 빠르게 쌓도록 구성했습니다.
- 현재 폴더는 사람마다 다를 수 있으니, 폴더 경로는 본인 환경 기준으로 보이면 정상입니다.

---

## 1단계. 파일 하나 만들어보기

### 나

```text
현재 폴더에 test.txt 파일 하나 만들어줘
```

### 코덱스

```text
현재 작업 폴더에 test.txt를 만들어서, 코텍스가 실제로 파일도 다룰 수 있다는 흐름을 바로 보여드릴게요.
```

### 포인트

- AI가 답만 하는 게 아니라 실제로 파일도 생성할 수 있다는 걸 체감하는 단계

---

## 2단계. 파이썬 코드 만들어보기

### 나

```text
파이썬으로 계산기 코드 만들어줘
```

### 코덱스

```text
현재 폴더에 아주 간단한 계산기 파이썬 파일을 만들어둘게요. 바로 실행해볼 수 있게 기본 사칙연산이 되는 형태로 넣겠습니다.
```

### 포인트

- 코드를 설명만 하는 게 아니라 실제 파일로 생성 가능
- 초보자는 `무에서 시작하기` 경험을 얻을 수 있음

---

## 3단계. 실행 방법 물어보기

### 나

```text
실행방법 알려줘
```

### 코덱스

```text
현재 폴더 기준으로 `python calculator.py` 또는 `py calculator.py`로 실행하면 됩니다.
```

### 포인트

- 코드를 만든 뒤 끝나는 게 아니라, 실행 방법까지 바로 이어서 배울 수 있음

---

## 4단계. 기능 하나 추가해보기

### 나

```text
나머지(%) 기능도 추가해줘
```

### 코덱스

```text
기존 계산기에 `%` 연산을 추가해서, 코덱스로 파일 수정까지 해볼 수 있게 진행하겠습니다.
```

### 포인트

- 기존 코드 수정 요청도 아주 자연스럽게 가능
- "새로 만들기"보다 더 실무에 가까운 경험

---

## 5단계. 코드 설명받기

### 나

```text
이 코드가 어떻게 동작하는지 설명해줘
```

### 코덱스

```text
함수 역할, 입력 흐름, 조건문, 마지막 실행 진입점까지 순서대로 설명해드릴게요.
```

### 포인트

- 코드를 만든 뒤 바로 학습 자료로 전환 가능
- 초보자 온보딩에 매우 강함

---

## 6단계. 에러 안 나게 개선하기

### 나

```text
숫자 말고 문자를 입력해도 안 죽게 개선해줘
```

### 코덱스

```text
입력값 검사를 추가해서 프로그램이 바로 종료되지 않도록 개선하겠습니다.
```

### 포인트

- "동작하는 코드"에서 "조금 더 쓸만한 코드"로 발전시키는 단계
- 실습 참가자가 개선 요청 감각을 익히기 좋음

---

## 7단계. 반복 실행되게 바꾸기

### 나

```text
반복해서 계속 계산할 수 있게 바꿔줘
```

### 코덱스

```text
여러 번 연속 계산하고, 원할 때 종료할 수 있게 반복 구조를 추가하겠습니다.
```

### 포인트

- 한 번 실행용 코드에서 실제 사용 가능한 형태로 개선
- 요구사항을 말로 전달해도 코드로 바뀌는 경험

---

## 8단계. 문서도 만들어보기

### 나

```text
사용법을 README로 정리해줘
```

### 코덱스

```text
지금 계산기 사용법을 정리한 README.md를 현재 폴더에 추가하겠습니다.
```

### 포인트

- 코드는 물론 문서 작성까지 한 흐름 안에서 가능
- 협업과 공유를 위한 준비까지 이어짐

---

## 9단계. 웹페이지로 바꿔보기

### 나

```text
이 계산기를 웹페이지로 만들어줘
```

### 코덱스

```text
브라우저에서 바로 열어 계산할 수 있게 HTML, CSS, JavaScript 한 파일로 만들어두겠습니다.
```

### 포인트

- 콘솔 프로그램을 웹 UI로 확장하는 경험
- 짧은 프롬프트만으로 결과물이 눈에 보이기 시작함

---

## 10단계. 버튼형 UI와 키보드 지원 추가

### 나

```text
버튼형 계산기로 바꿔줘
```

### 코덱스

```text
숫자 버튼을 눌러 식을 만들고 계산하는 방식으로 웹 계산기를 바꾸겠습니다.
```

### 나

```text
키보드 입력도 되게 해줘
```

### 코덱스

```text
숫자키, 연산자키, Enter, Backspace, Escape까지 동작하게 수정하겠습니다.
```

### 포인트

- UI 수정과 동작 개선이 모두 가능함
- 실습 참가자가 "이제 진짜 앱처럼 보인다"는 느낌을 받기 좋은 단계

---

## 마무리 실습. GitHub에 올릴 수 있게 정리해줘

### 나

```text
깃허브에 올릴수 있게 정리해줘
```

### 코덱스

```text
README, .gitignore, 저장소 초기화, 스테이징까지 정리해서 GitHub에 올리기 직전 상태로 맞춰드릴게요.
```

### 포인트

- 코딩 실습에서 끝나지 않고 공유 가능한 결과물까지 연결됨
- 실제 협업 흐름과 가장 가까운 마무리

---

## 특강 진행 팁

- 처음 10분은 "말로 시키면 파일이 생긴다"는 경험을 주는 데 집중합니다.
- 중간 20분은 "기능 추가와 수정"을 보여주면 가장 반응이 좋습니다.
- 후반 20분은 "웹페이지로 확장"하면 눈에 보이는 변화가 커서 집중도가 올라갑니다.
- 마지막 10분은 "문서화 + GitHub 정리"로 실무 연결감을 주면 좋습니다.
