# ChatGPT Enterprise / GPTs / Codex 핸즈온 실습

## 실습 안내

이 문서는 **해외지원그룹 AI PoC 인력 대상 핸즈온 실습**을 위해 준비한 **원페이지 실습 자료**입니다.

오늘 실습은 여러 페이지를 왔다 갔다 하지 않고, **이 페이지 하나를 띄워놓고 순서대로 함께 따라가는 방식**으로 진행합니다.

각자 PC에서 이 페이지를 열어둔 상태로 **예시 프롬프트를 복사/붙여넣기** 하며 실습하시면 됩니다.

---

## 오늘 실습 목표

오늘 실습의 목표는 아래 4가지입니다.

- ChatGPT Enterprise의 핵심 기능을 업무 관점에서 이해한다.
- GPTs를 활용해 간단한 에이전트형 PoC를 만드는 흐름을 익힌다.
- Codex가 어떤 작업에 강한지 감을 잡는다.
- 해외지원그룹 업무에 연결할 수 있는 실습 아이디어를 얻는다.

---

## 오늘 전체 실습 구성

오늘 실습은 아래 **3개 큰 섹션**으로 진행합니다.

1. **ChatGPT Enterprise, 바로 쓰는 업무 활용**
2. **GPTs, 업무형 에이전트 PoC 같이 만들어보기**
3. **Codex, 실행형 자동화와 코딩 에이전트 맛보기**

# 1. ChatGPT Enterprise, 바로 쓰는 업무 활용

## 1-1. 이 섹션은 이렇게 진행합니다

ChatGPT Enterprise는 가장 범용적으로 업무에 활용할 수 있는 도구입니다.

메일 초안 작성, 문장 다듬기, 요약, 조사, 자료 정리, 이미지 생성 등 다양한 작업에 활용할 수 있습니다.

### 프로젝트 / 파일 업로드 / RAG

- 프로젝트는 관련 자료와 대화를 한곳에 모아 관리할 수 있는 작업 공간입니다.

- RAG는 문서 기반으로 답변 품질과 근거성을 높이는 방식입니다.
- 오늘 뒤에서 다룰 GPTs 실습과도 자연스럽게 연결됩니다.

# 2. GPTs, 업무형 에이전트 PoC 같이 만들어보기

## 2-1. GPTs란

GPTs는 단순 대화형 챗봇이 아니라, **지침 + 지식 + 출력 형식**을 설계해 특정 업무를 수행하도록 만든 맞춤형 에이전트입니다.

핵심은 아래 3가지입니다.
- 어떤 역할을 맡길 것인가
- 어떤 문서를 참고하게 할 것인가
- 어떤 형식으로 결과를 내게 할 것인가

### GPTs에 들어갈 항목 예시

아래 내용은 ChatGPT로 실제 생성한 예시이며, GPTs 생성 화면에 그대로 복사/붙여넣기 해서 사용할 수 있습니다.

#### 1. 이름

```text
AI Crew 운영 도우미
```

#### 2. 설명

```text
AI Crew 비용 정산, 계정, 신청, 운영 기준 등과 관련된 질문에 대해 업로드된 가이드 문서를 기반으로 빠르고 정확한 1차 답변을 제공합니다.
문서에 없는 내용은 추측하지 않고, 필요한 경우 담당 부서 및 문의 경로를 안내합니다.
```

#### 3. 지침

```text
역할
당신은 “AI Crew 운영 도우미”입니다.
AI Crew 운영과 관련된 실무 질문에 대해 업로드된 문서를 근거로 1차 답변을 제공합니다.

답변 원칙

문서 우선
반드시 업로드된 아래 문서를 먼저 참고하여 답변합니다.
- AI Crew 비용 정산 가이드
- AI Crew 역할 안내 상세 자료

추측 금지
문서에 없는 내용은 절대 추측하지 않습니다.
불확실한 경우 “문서 기준으로 확인되지 않음”이라고 명확히 표현합니다.

문의 경로 안내
문서에 없는 경우 아래 형태로 안내합니다.
- 해당 내용은 가이드에 명시되어 있지 않습니다.
- 정확한 확인을 위해 담당 부서 또는 담당자에게 문의해 주세요.

간결하고 실무적으로
답변은 최대한 짧고 명확하게 작성합니다.
불필요한 설명은 제외하고, 바로 실행 가능한 정보 중심으로 제공합니다.

구조화된 답변
가능하면 아래 형식을 사용합니다.
- ✔️ 핵심 답변
- ✔️ 근거 (문서 기준)
- ✔️ 추가 안내 (필요 시)

FAQ 제안
질문과 관련된 유사 질문이 있을 경우 아래 형식으로 제안합니다.
- 참고 FAQ:
- ○○는 어떻게 하나요?
- △△ 기준은 무엇인가요?

답변 예시 스타일

✔️ 핵심 답변
AI Crew 비용 정산은 월 단위로 진행됩니다.

✔️ 근거
비용 정산 가이드에 따르면 월 마감 기준으로 정산 처리됩니다.

✔️ 추가 안내
```

### 해외지원그룹에서 활용해볼 만한 RAG 챗봇 아이디어

#### 1. 해외지원 운영 가이드 챗봇
- 해외지원그룹 내부 운영 가이드, FAQ, 프로세스 문서를 붙여두고 반복 질문에 1차 답변하는 형태
- 신규 인력 온보딩이나 자주 들어오는 문의 대응에 활용 가능

#### 2. 법인 문의 분류 및 답변 도우미 챗봇
- 해외 법인에서 자주 들어오는 문의 유형, 처리 기준, 담당 부서를 문서화해두고 빠르게 안내하는 형태
- “이 문의는 어디로 보내야 하지?” 같은 상황에서 바로 써먹기 좋음

#### 3. 정산 / 신청 / 운영 기준 안내 챗봇
- 비용 정산, 계정 신청, 운영 기준, 제출 항목 같은 문서를 모아두고 실무자가 바로 확인할 수 있게 하는 형태
- 메일이나 메신저로 반복 설명하는 시간을 줄이는 데 도움 가능

이런 아이디어는 완벽한 시스템으로 시작하기보다, **자주 묻는 질문이 많은 문서부터 작게 붙여서 PoC로 검증해보는 방식**이 좋습니다.

# 3. Codex, 실행형 자동화와 코딩 에이전트 맛보기

## 3-1. Codex란

Codex는 단순히 답변을 주는 챗봇이 아니라, **파일 생성, 코드 작성, 수정 방향 제안, 반복 작업 자동화**까지 도와주는 코딩 에이전트형 도구입니다.

## 3-2. ChatGPT vs Codex

- ChatGPT는 질문하고 답을 받는 데 강합니다.
- Codex는 실제 작업 결과물을 만드는 데 더 강합니다.
- 둘은 경쟁 관계라기보다 역할이 다른 도구로 이해하면 좋습니다.

## 3-3. 네비게이션 vs 자율주행

비유하면,
- ChatGPT는 네비게이션
- Codex는 자율주행

즉, ChatGPT는 방향을 잘 잡아주고, Codex는 실제로 손이 많이 가는 작업을 더 적극적으로 수행하는 느낌으로 이해하시면 됩니다.

## 3-4. Codex 실행하는 3가지 방법

- CLI
- App
- IDE

### 참고 포인트

- 초보자는 App 방식이 상대적으로 진입장벽이 낮습니다.
- 익숙한 사람은 CLI나 IDE 연동이 더 강력할 수 있습니다.

## 3-5. Codex가 강한 작업 예시

- 파일 생성
- 다중 파일 수정
- 간단한 프로그램 만들기
- 자동화 스크립트 작성

- 메일 자동화 코드 만들기
- 반복 작업 정리용 스크립트 만들기
- 엑셀 병합 스크립트 작성
- 입력/처리/출력 구조가 있는 업무 자동화
- 나중에 재사용 가능한 스킬 형태로 정리

# 마무리

오늘 실습에서 가장 중요한 메시지는 아래와 같습니다.

- ChatGPT Enterprise는 바로 쓰는 업무 생산성 도구입니다.
- GPTs는 지식, 지침, 출력 형식을 설계해서 업무형 에이전트를 빠르게 만드는 도구입니다.
- Codex는 실제 파일 생성, 코드 작성, 자동화 초안 작성에 강한 도구입니다.
- PoC는 크게 시작하는 것이 아니라, 작은 실습에서 바로 검증해보는 것이 중요합니다.

한 줄로 정리하면 이렇습니다.

**ChatGPT Enterprise, GPTs, Codex는 반복 업무를 줄이고, 문서와 지식을 활용한 실무형 PoC를 빠르게 검증하는 데 매우 유용한 도구입니다.**
