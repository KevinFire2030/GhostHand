# 오프닝

안녕하세요.
오늘은 GPTs를 활용해서 AX PM 그룹 관점의 AX 과제 PoC를 어떻게 빠르게 만들어볼 수 있는지 30분 정도 실습 중심으로 같이 보겠습니다.

오늘 강의의 핵심은 복잡한 이론 설명이 아닙니다.
실제로 GPTs를 어떻게 구성하고, 어떤 식으로 지식과 지침을 넣고, 어떤 수준까지 PoC를 빠르게 만들어볼 수 있는지 직접 확인하는 것이 목적입니다.

특히 AX PM 업무에서는
- 문서 기반 질의응답
- 외부 뉴스 기반 보고서 초안 작성
- 정형 데이터 기반 분석
- 외부 API 연동 자동화
같은 형태의 PoC가 꽤 자주 필요합니다.

오늘은 이걸 GPTs로 얼마나 빠르게 구현할 수 있는지 예시를 통해 보여드리겠습니다.

강의는 `docs/course/chatgpt_gemini` 폴더를 기준으로 진행하겠습니다.
이 폴더 안에 정리해둔 PoC 생성 가이드를 하나씩 보면서,
직접 GPT를 만드는 방식으로 실습을 진행하겠습니다.

오늘은 완성형 서비스를 만드는 시간이 아니라,
업무 적용 가능성을 빠르게 검증하는 PoC를 어떻게 만들 수 있는지 보는 시간이라고 생각해주시면 좋겠습니다.

그리고 기본적인 사용법은 아래 핸드온 교육 영상을 함께 참고하시면 좋겠습니다.

- ChatGPT Enterprise Hands-On 교육 - 녹스미팅 녹화본(26.4.22)
  - https://mosaic.sec.samsung.net/collaboration/workspace/5000000237/list/5000006164/detail/5000717805
- 비개발자를 위한 Agent 맛보기 교육 - 녹스미팅 녹화본(26.4.22)
  - https://mosaic.sec.samsung.net/collaboration/workspace/5000000237/list/5000006164/detail/5000721765

특히 이번에 이미지 2.0 소개는 꽤 인상적이었습니다.
- 기존에 한글 깨지는 문제가 거의 해결된 수준으로 보였고,
- 실제로 써보면 활용도가 높겠다는 생각이 들었습니다.
- 꼭 한 번씩 써보시길 권장드립니다.

그럼 먼저 전체 구조부터 간단히 보겠습니다.
