# GPTs_PoC3

## 1. PRD

### (결산P) 손익 분석 자동화

- 법인별 총원가표 구성이 목적이며 원가DB를 기반으로 전월/전년/동기비 판가차, 물량차, O/H차를 분석하여 워드 보고서를 작성하는 내용으로
- PoC 버전은 업로드의 한계가 있으니 샘플 데이터를 구성해서 PoC하되 각 컬럼에 대한 정의가 아주 명확해야 하고 특히 산출식은 실제 엑셀에서 수식을 정의하듯이 학습시켜야 함 (팀장님 코멘트)
- 현재 메디슨에서 구글AI를 사용하고 있다고 하니 내용을 들어보고 필요하면 동일하게 적용하는것을 검토해볼것

## 2. GPTs 단계별 구현 시나리오

### STEP_1. 목적 정의
- 손익 분석 자동화 PoC의 목표를 명확히 정의합니다.
- 최종 산출물이 무엇인지 정리합니다.
- 예: 법인별 총원가표 기반 손익 분석 결과와 워드 보고서 초안 생성

### STEP_2. 입력 데이터 구조 정의
- 원가DB에서 어떤 컬럼이 필요한지 정의합니다.
- 전월, 전년, 동기비 비교를 위해 필요한 기준 항목을 정리합니다.
- 판가차, 물량차, O/H차 계산에 필요한 입력값을 명확히 분리합니다.

### STEP_3. 샘플 데이터 구성
- PoC용 샘플 데이터를 설계합니다.
- 실제 업로드 한계를 고려해 대표성 있는 샘플만 선정합니다.
- 각 컬럼의 의미와 값의 예시를 함께 정의합니다.

### STEP_4. 산출식 정의
- 판가차, 물량차, O/H차 계산 공식을 정의합니다.
- 실제 엑셀 수식을 만들듯이 계산 로직을 명확하게 문서화합니다.
- 사용자가 봐도 이해할 수 있게 항목별 계산 기준을 설명합니다.

### STEP_5. GPTs 프롬프트 설계
- GPTs가 어떤 역할을 해야 하는지 정의합니다.
- 예: 데이터 해석, 차이 분석, 보고서 문장 생성
- 입력값과 출력 형식을 표준화합니다.

### STEP_6. 분석 결과 생성
- 샘플 데이터를 입력했을 때 GPTs가 손익 분석 결과를 작성하도록 합니다.
- 차이 원인을 항목별로 요약하게 합니다.
- 핵심 변화 포인트를 경영진 관점에서 읽기 쉽게 정리하도록 합니다.

### STEP_7. 워드 보고서 초안 생성
- 분석 결과를 바탕으로 워드 보고서 형식의 문서 초안을 생성합니다.
- 보고서 구조는 개요, 주요 분석 결과, 차이 원인, 시사점 순으로 정리합니다.
- 사람이 후편집하기 쉽도록 문단 중심으로 작성합니다.

### STEP_8. 검증 및 개선
- 샘플 데이터 기준으로 결과 정확성을 검토합니다.
- 계산식, 컬럼 정의, 문장 표현을 수정합니다.
- 실제 현업 피드백을 반영해 PoC 완성도를 높입니다.

### STEP_9. 운영 적용 검토
- 메디슨의 구글AI 사용 사례를 확인합니다.
- 필요 시 동일하거나 유사한 방식으로 적용 가능한지 검토합니다.
- 향후 실제 운영 전환 시 필요한 조건을 정리합니다.

법인별 총원가표 기반 손익 분석 결과와 워드 보고서 초안 생성

### STEP_2. 입력 데이터 구조 정의
원가DB에서 어떤 컬럼이 필요한지 정의합니다.  
전월, 전년, 동기비 비교를 위해 필요한 기준 항목을 정리합니다.  
판가차, 물량차, O/H차 계산에 필요한 입력값을 명확히 분리합니다.

(산출물 예시)  
샘플 원가DB 컬럼 정의
- 법인코드
- 기준월
- 전월매출액
- 당월매출액
- 전년동월매출액
- 판매수량
- 전월판매수량
- 전년동월판매수량
- 재료비
- 노무비
- 경비
- O/H
- 판가
- 전월판가
- 전년동월판가

### STEP_3. 샘플 데이터 구성
PoC용 샘플 데이터를 설계합니다.  
실제 업로드 한계를 고려해 대표성 있는 샘플만 선정합니다.  
각 컬럼의 의미와 값의 예시를 함께 정의합니다.

(산출물 예시)  
- STEP_2에서 정의한 샘플 원가DB 컬럼 기준으로 더미 데이터 100개 생성  
- 법인별, 월별, 품목별 데이터 분포가 보이도록 샘플 구성  
- 샘플 데이터 파일명: `샘플 데이터 샘플.xlsx`

### STEP_4. 산출식 정의
판가차, 물량차, O/H차 계산 공식을 정의합니다.  
실제 엑셀 수식을 만들듯이 계산 로직을 명확하게 문서화합니다.  
사용자가 봐도 이해할 수 있게 항목별 계산 기준을 설명합니다.

(산출물 예시)  
- 판가차 = (당월판가 - 전월판가) × 당월판매수량  
- 물량차 = (당월판매수량 - 전월판매수량) × 전월판가  
- O/H차 = 당월O/H - 전월O/H  
- 전년동기비 차이 = 당월값 - 전년동월값
