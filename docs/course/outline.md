# 강의 전체 구조

## 강의 방향
- 책의 핵심 메시지를 강의형으로 재구성한다.
- 개념 설명보다 실제 구축과 적용 경험에 초점을 둔다.
- 참가자가 "나도 해볼 수 있다"는 감각을 얻도록 설계한다.

## 대상
- AI 에이전트와 자동화에 관심 있는 개인
- OpenClaw 기반 모바일 에이전트 구축에 관심 있는 실무자
- 업무/일상 자동화를 고민하는 기획자, 개발자, PM

## 강의 목표
- 모바일 AI 에이전트 개념 이해
- OpenClaw 기반 전체 구조 이해
- 실제 활용 시나리오 이해
- 개인/업무 적용 아이디어 도출

## 권장 구성안
### 1안. 90분 특강
- 모바일 AI 에이전트란 무엇인가
- OpenClaw와 GhostHand 소개
- 주요 데모
- 적용 사례
- Q&A

### 2안. 3~4시간 워크숍
- 환경 구성
- 기본 연결
- 첫 성공 경험 만들기
- 간단한 자동화 실습

### 3안. 4주 프로그램
- 1주차: 개념과 환경 구축
- 2주차: 에이전트, 메모리, 툴
- 3주차: 자동화 시나리오 설계
- 4주차: 개인 프로젝트 적용

## 추후 정리할 것
- 강의 대상별 버전 분화
- 실습 난이도 구분
- 데모 체크리스트
- 홍보 문구 연결
