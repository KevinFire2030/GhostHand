# 02. 이미지 실습

## 실습 목표

이 실습은 ChatGPT의 **이미지 생성 및 이미지 기반 업무 활용 방식**을 익히는 것을 목표로 합니다.

교육생은 아래를 직접 경험합니다.

- 텍스트만으로 원하는 이미지를 요청하는 방법
- 업무 목적에 맞게 이미지 스타일을 구체화하는 방법
- 생성된 결과를 바탕으로 다시 수정 요청하는 방법
- 단순 재미용 이미지가 아니라 **업무 설명, 문서, 보고, 아이디어 전달용 이미지**로 활용하는 방법

## 실습 시나리오

해외지원팀 업무에서는 이미지가 꼭 디자인 부서 전용 결과물일 필요는 없습니다.

예를 들면 아래와 같은 활용이 가능합니다.

- 발표 자료에 들어갈 개념 설명용 이미지 만들기
- 업무 프로세스를 직관적으로 설명하는 삽화 만들기
- 고객 설명용 예시 화면이나 분위기 이미지 만들기
- 교육 자료 표지 이미지 만들기
- 보고서, 제안서, 사내 공유자료에 넣을 시각 자료 초안 만들기

이 실습에서는 “예쁘게 만드는 것”보다, **업무 맥락에 맞는 이미지를 빠르게 만들고 수정하는 경험**에 집중합니다.

## 진행 방법

1. ChatGPT에서 새 채팅을 엽니다.
2. 이미지 생성이 가능한 상태인지 확인합니다.
3. 아래 예시 프롬프트 중 하나를 복사해서 붙여 넣습니다.
4. 생성된 이미지를 확인합니다.
5. 마음에 들지 않으면 수정 요청을 추가합니다.

예를 들면 아래처럼 다시 요청할 수 있습니다.

```text
좀 더 깔끔하고 발표 자료용 느낌으로 바꿔줘.
```

```text
배경은 단순하게 하고 핵심 요소만 남겨줘.
```

```text
좀 더 비즈니스 문서에 어울리는 톤으로 수정해줘.
```

```text
텍스트가 들어가지 않게 다시 만들어줘.
```

## 예시 프롬프트 1. 발표 자료용 표지 이미지

상황: 해외지원팀 AI 교육 자료 첫 장에 넣을 표지 이미지가 필요한 경우

```text
해외지원팀의 AI 업무 활용 교육 자료 표지에 어울리는 이미지를 만들어줘.

조건:
- 전체 분위기는 깔끔하고 전문적
- AI, 글로벌 협업, 업무 생산성 향상 느낌이 함께 보이면 좋음
- 발표 자료 첫 장에 들어갈 수 있는 스타일
- 텍스트는 이미지 안에 넣지 말 것
- 너무 화려하지 않고 회사 교육자료에 어울리게
```

## 예시 프롬프트 2. 업무 프로세스 설명용 이미지

상황: 업무 흐름을 직관적으로 설명하고 싶은 경우

```text
아래 업무 흐름을 한눈에 이해할 수 있는 일러스트 스타일 이미지로 만들어줘.

업무 흐름:
- 해외 법인에서 문의 접수
- 본사 지원팀에서 내용 검토
- 관련 부서 협의
- 결과 회신

조건:
- 발표 자료에 넣기 좋게 단순하고 명확한 구성
- 아이콘이나 흐름 느낌이 잘 보이게
- 복잡한 텍스트는 넣지 말 것
- 비즈니스 문서에 어울리는 색감
```

## 예시 프롬프트 3. 고객 설명용 개념 이미지

상황: RAG 기반 챗봇 개념을 쉽게 설명하고 싶은 경우

```text
RAG 기반 챗봇의 개념을 비전문가도 직관적으로 이해할 수 있게 설명하는 이미지를 만들어줘.

조건:
- 사용자의 질문이 들어오고
- 사내 문서 또는 지식 저장소를 참고한 뒤
- 답변이 생성되는 흐름이 느껴지게
- 너무 기술 다이어그램처럼 어렵지 않게
- 교육 자료에 넣기 좋은 깔끔한 스타일
- 텍스트는 최소화하거나 넣지 말 것
```

## 예시 프롬프트 4. 보고서 삽입용 비주얼 생성

상황: 보고서에 넣을 AI 활용 분위기 이미지가 필요한 경우

```text
기업의 실무자가 AI를 활용해 업무를 처리하는 장면을 보고서 삽입용 이미지로 만들어줘.

조건:
- 과장된 미래도시 느낌 말고 현실적인 사무 환경
- 노트북, 대시보드, 문서, 협업 느낌이 자연스럽게 보일 것
- 전체적으로 신뢰감 있고 차분한 분위기
- 회사 보고서에 넣어도 어색하지 않게
- 텍스트는 넣지 말 것
```

## 예시 프롬프트 5. 교육 안내 자료용 이미지

상황: 실습 교육 안내 페이지나 포스터에 쓸 이미지가 필요한 경우

```text
ChatGPT와 Codex를 활용한 실무형 핸즈온 교육을 안내하는 이미지를 만들어줘.

조건:
- 교육, 실습, 디지털 업무도구, 생산성 향상 느낌
- 초보자도 부담 없이 참여할 수 있는 친근한 분위기
- 동시에 전문성도 느껴지게
- 사내 교육 포스터나 안내 페이지에 활용 가능한 스타일
- 텍스트는 이미지 안에 넣지 말 것
```

## 실습 후 추가 요청 예시

첫 결과를 받은 뒤에는 아래처럼 수정 요청을 하면서 품질을 높일 수 있습니다.

```text
좀 더 미니멀한 스타일로 다시 만들어줘.
```

```text
색감을 파란색 계열 중심의 비즈니스 톤으로 맞춰줘.
```

```text
발표 자료에 넣을 거라서 요소를 줄이고 더 단정하게 정리해줘.
```

```text
사람이 너무 많이 나오지 않게 해줘.
```

```text
아이콘형 인포그래픽 느낌으로 다시 바꿔줘.
```

## 실습 포인트

이 실습에서 중요한 포인트는 “한 번에 완벽한 이미지를 만드는 것”이 아닙니다.

핵심은 아래와 같습니다.

- 목적을 먼저 분명하게 말한다.
- 스타일, 분위기, 사용처를 함께 알려준다.
- 첫 결과를 본 뒤 수정 지시를 추가한다.
- 점점 업무에 맞는 방향으로 다듬는다.

즉, 이미지 기능도 단발성 생성보다는 **업무 결과물을 반복 보정하는 협업 도구**처럼 쓰는 것이 좋습니다.

## 강사 진행 멘트 예시

- 이번에는 이미지 기능을 실습해보겠습니다.
- 중요한 건 그림을 잘 그리는 게 아니라, 업무 목적에 맞는 이미지를 빠르게 만드는 경험입니다.
- 발표 자료, 보고서, 설명자료에 넣을 수 있는 이미지도 충분히 만들 수 있습니다.
- 먼저 하나를 복사해서 붙여 넣고, 그 다음 수정 요청으로 원하는 방향에 맞춰보겠습니다.

## 기대 효과

이 실습이 끝나면 교육생은 아래를 할 수 있어야 합니다.

- 텍스트로 원하는 이미지 결과를 요청할 수 있다.
- 업무 맥락에 맞는 이미지 스타일을 구체적으로 지시할 수 있다.
- 결과를 보고 수정 요청을 통해 품질을 높일 수 있다.
- 교육자료, 보고서, 설명자료에 이미지 기능을 실무적으로 활용할 수 있다.
