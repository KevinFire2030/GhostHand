# 03. 심층 리서치 실습

## 실습 목표

이 실습은 ChatGPT의 **심층 리서치(Deep Research)** 기능을 활용해, 단순 검색이 아니라 **업무 의사결정에 도움이 되는 조사 결과**를 얻는 방법을 익히는 것을 목표로 합니다.

교육생은 아래를 직접 경험합니다.

- 조사 목적을 분명하게 정의하는 방법
- 리서치 범위와 출력 형식을 함께 요청하는 방법
- 단순 요약이 아니라 비교, 정리, 시사점까지 끌어내는 방법
- 해외지원팀 업무에 연결될 수 있는 조사형 활용 사례를 익히는 방법

## 실습 시나리오

해외지원팀 업무에서는 아래와 같은 조사형 요청이 자주 발생할 수 있습니다.

- 특정 시장이나 지역의 동향을 빠르게 파악해야 한다.
- 경쟁사 사례나 벤치마킹 포인트를 정리해야 한다.
- 여러 자료를 읽고 핵심만 압축한 브리핑이 필요하다.
- 새로운 솔루션, 서비스, 정책 변화 등을 검토해야 한다.
- 회의 전에 배경지식과 주요 쟁점을 빠르게 정리해야 한다.

이럴 때 심층 리서치는 단순 검색 결과를 나열하는 것이 아니라, **업무용 조사 메모나 브리핑 초안**을 얻는 데 강점이 있습니다.

## 진행 방법

1. ChatGPT에서 새 채팅 또는 심층 리서치 기능을 사용할 수 있는 화면을 엽니다.
2. 아래 예시 프롬프트 중 하나를 복사해서 붙여 넣습니다.
3. 조사 결과가 나오면 구조와 깊이를 확인합니다.
4. 부족한 부분이 있으면 범위, 대상, 형식을 다시 지정합니다.
5. 필요하면 후속 질문으로 더 구체화합니다.

예를 들어 아래처럼 추가 요청할 수 있습니다.

```text
해외지원팀 관점에서 실무 시사점만 다시 정리해줘.
```

```text
표로 비교할 수 있게 다시 구성해줘.
```

```text
출처를 포함해서 핵심만 1페이지 분량으로 줄여줘.
```

```text
임원 보고용으로 더 간결하게 요약해줘.
```

## 예시 프롬프트 1. AI 도입 사례 조사

상황: 다른 기업들이 생성형 AI를 어떻게 업무에 적용하는지 파악하고 싶은 경우

```text
국내외 기업들이 생성형 AI를 실제 업무에 도입한 사례를 조사해줘.

조건:
- 고객지원, 사내지식검색, 문서작성, 보고서 자동화 중심
- 국내 사례와 해외 사례를 구분해서 정리
- 각 사례별로 무엇을 자동화했는지, 기대효과는 무엇인지 요약
- 마지막에는 우리 회사 해외지원팀 관점에서 참고할 만한 시사점 5가지를 정리
```

## 예시 프롬프트 2. 경쟁 서비스 비교 조사

상황: 도입 검토 중인 서비스들을 비교해야 하는 경우

```text
ChatGPT Enterprise, Microsoft Copilot, Google Gemini for Workspace를 업무 활용 관점에서 비교 조사해줘.

조건:
- 비교 항목: 문서작성 지원, 데이터 활용, 협업성, 보안/관리, 사용자 편의성
- 표로 먼저 정리
- 그 다음 각 서비스의 강점과 약점을 요약
- 마지막에는 해외지원팀의 실무 적용 관점에서 어떤 상황에 어떤 도구가 적합할지 의견 제시
```

## 예시 프롬프트 3. 특정 국가 또는 지역 시장 조사

상황: 특정 지역의 산업 또는 고객 환경을 빠르게 파악해야 하는 경우

```text
동남아 지역에서 기업용 AI 업무도구 도입 트렌드를 조사해줘.

조건:
- 최근 동향 중심
- 주요 국가 몇 곳을 나눠서 설명
- 어떤 산업군에서 도입이 활발한지 정리
- 실제 업무에 어떤 방식으로 적용되는지 사례가 있으면 포함
- 해외지원팀이 참고할 만한 포인트를 마지막에 정리
```

## 예시 프롬프트 4. 정책/규제 변화 조사

상황: AI 활용과 관련된 정책이나 규제 변화를 파악해야 하는 경우

```text
기업의 생성형 AI 활용과 관련해 최근 주요 국가 또는 주요 시장에서 논의되는 정책, 규제, 가이드라인 이슈를 조사해줘.

조건:
- 너무 법률 전문 문체보다는 실무자가 이해할 수 있게 설명
- 기업 입장에서 주의할 점 중심으로 정리
- 보안, 개인정보, 저작권, 내부 통제 관점 포함
- 마지막에는 기업 실무자가 체크해야 할 항목을 체크리스트 형태로 제시
```

## 예시 프롬프트 5. 회의 전 브리핑 자료 만들기

상황: 회의 전에 빠르게 배경지식을 정리해야 하는 경우

```text
아래 주제로 10분 안에 읽을 수 있는 회의 전 브리핑 문서를 만들어줘.

주제:
해외지원 조직에서 생성형 AI를 활용해 반복 업무를 줄이는 방안

조건:
- 배경
- 주요 활용 영역
- 기대효과
- 리스크
- 먼저 작게 시작할 수 있는 PoC 아이디어
- 경영진에게 설명할 때 사용할 수 있는 핵심 메시지 3개 포함
```

## 실습 후 추가 요청 예시

리서치 결과를 받은 뒤, 아래처럼 후속 요청하면 더 실무적인 결과로 발전시킬 수 있습니다.

```text
핵심만 남기고 경영진 보고용 1페이지 요약으로 바꿔줘.
```

```text
해외지원팀 관점에서 당장 실행 가능한 아이디어만 추려줘.
```

```text
표, bullet, 결론 형식으로 다시 정리해줘.
```

```text
조사 결과 중 불확실하거나 추가 확인이 필요한 부분도 따로 표시해줘.
```

```text
이 내용을 바탕으로 발표자료 5장 분량 목차도 만들어줘.
```

## 실습 포인트

이 실습에서 중요한 포인트는 “많이 찾는 것”이 아니라 **의사결정에 쓸 수 있게 정리하는 것**입니다.

핵심은 아래와 같습니다.

- 조사 대상과 목적을 분명히 한다.
- 어떤 형식으로 받고 싶은지 함께 요청한다.
- 단순 정보 나열이 아니라 비교, 시사점, 추천 형태로 발전시킨다.
- 결과를 그대로 믿기보다 필요한 경우 추가 검증 포인트도 확인한다.

즉, 심층 리서치는 검색 대체가 아니라 **업무 조사 보조자**처럼 쓰는 것이 가장 효과적입니다.

## 강사 진행 멘트 예시

- 이번에는 심층 리서치 기능을 실습해보겠습니다.
- 단순히 검색 결과를 많이 보는 것이 아니라, 업무에 쓸 수 있는 조사 결과를 빠르게 얻는 게 핵심입니다.
- 질문할 때 조사 범위와 원하는 결과 형식을 같이 주면 훨씬 좋은 결과가 나옵니다.
- 먼저 예시를 하나 실행해보고, 그 다음 후속 요청으로 결과를 더 실무적으로 다듬어보겠습니다.

## 기대 효과

이 실습이 끝나면 교육생은 아래를 할 수 있어야 합니다.

- 조사형 질문을 업무 목적에 맞게 구조화할 수 있다.
- 여러 자료를 빠르게 요약, 비교, 정리한 결과를 받아볼 수 있다.
- 시장조사, 벤치마킹, 정책 파악, 회의 준비 등에 AI를 활용할 수 있다.
- 조사 결과를 보고서, 발표자료, PoC 기획 초안으로 연결할 수 있다.
