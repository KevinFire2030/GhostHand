# Overseas Support Hands-on Course

## 과정 개요

이 폴더는 **해외지원팀의 AI PoC 인력 대상** 실습 교육 자료를 저장하는 공간입니다.

회사가 도입을 검토 중인 **ChatGPT Enterprise**와 **Codex**를 중심으로, 실제 업무에 바로 연결할 수 있는 활용 사례를 핸즈온 데모 형태로 실습하는 것을 목표로 합니다.

- 대상: 해외지원팀 AI PoC 인력
- 목적: 업무 활용 사례 중심 실습 교육 자료 저장 및 공유
- 예상 진행 시간: 약 1시간
- 진행 방식: 강사 안내 + 교육생 개별 브라우저 실습

## 교육 진행 방식

교육 시 다음과 같은 방식으로 진행합니다.

- 강사는 **PC와 TV를 연결**합니다.
- 강사는 **웹브라우저로 이 페이지를 띄워놓고** 실습을 안내합니다.
- 교육생은 **각자의 PC에서 웹브라우저로 이 페이지를 띄워놓고** 실습에 참여합니다.
- 교육생은 설명을 들으며 **예시 프롬프트를 복사/붙여넣기** 하는 방식으로 실습을 진행합니다.
