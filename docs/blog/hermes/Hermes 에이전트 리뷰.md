### 1) 요약

이 영상은 OpenClaw 사용자 관점에서 Hermes 에이전트를 설치하고 실제로 설정해 보며 기능을 리뷰한다. Hermes는 스스로 진화하는 학습형 에이전트를 표방하며, 메모리 고도화, 스킬 자동화, 유연한 모델·백엔드 확장을 강점으로 소개된다.

주요 내용은 다음과 같다.

- OpenClaw의 기본 기능을 간단히 짚고, Hermes가 왜 주목받는지 설명한다.
- Hermes는 GitHub 스타 약 13만 개를 받은 프레임워크로 소개되며, 빠른 설치와 Android Termux 설치도 가능하다고 설명한다.
- 설치 과정에서 OpenClaw 설치가 감지되면 SOUL 문서, 기존 스킬, 데일리 메모리, 설정, 토큰 등을 Hermes 환경으로 마이그레이션할 수 있다.
- Hermes에는 약 670개의 빌트인 스킬이 있으며, Claude Code, Codex, OpenCode 등을 ACP 형태로 연결할 수 있다고 소개한다.
- File AI 기반 이미지·비디오 생성, ComfyUI, 디자인.md 기반 브랜드 스펙 설정 같은 스킬 활용 예시를 보여준다.
- OpenAI Codex 모델을 설정하고, 비용 효율을 위해 mini 모델을 일반 라우팅 용도로 쓰는 아이디어를 언급한다.
- Telegram 봇을 BotFather로 생성하고 토큰을 Hermes에 연결해 메시징을 설정하는 과정을 실습한다.
- Telegram을 홈 채널로 설정하면 Cron 결과, 공지, 다른 플랫폼 메시지 등을 텔레그램에서 받아볼 수 있다고 설명한다.
- LLM Wiki(KepaSEE)는 읽은 정보를 문서화하고 서로 연결해 시간이 지날수록 성장하는 지식 그래프를 만드는 방식으로 소개된다.
- 전체적으로 Hermes는 단순 챗봇이 아니라, OpenClaw의 장점을 이어받으면서 스킬·메모리·지식 그래프를 통해 성장하는 개인 AI 세컨브레인으로 평가된다.
