# A-3. 개인용 30일 전환 로드맵

개인의 에이젠틱 전환은 작게 시작해 점진적으로 확장하는 것이 좋다.

## 1주차: 반복 찾기
- 매일 반복되는 귀찮은 작업 5개 적기
- 가장 자주 반복되는 루틴 1~2개 선정

## 2주차: 요약 자동화
- 메일 요약
- 회의록 자동화
- 뉴스/정보 브리핑 만들기

## 3주차: 지식 연결
- 자주 쓰는 문서 정리
- RAG 또는 문서 검색 구조 붙이기
- 메신저나 Web UI로 연결하기

## 4주차: 실행 실험
- 일정 정리 자동화
- 아침/밤 루틴 자동화
- 승인 기반 실행 흐름 설계

핵심은 완벽함보다 지속 가능한 작은 성공을 쌓는 것이다.
