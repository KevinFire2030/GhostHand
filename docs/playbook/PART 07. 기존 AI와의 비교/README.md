# PART 07. 기존 AI와의 비교

이 파트는 스마트폰 제조사 내장 AI와 사용자 주도형 모바일 AI 에이전트를 직접 비교한다.  
앞선 파트들에서 모바일 AI 에이전트의 개념, 구조, 자동화, 실행 계층을 살펴봤다면, PART 7은 가장 직접적인 질문에 답한다.

핵심 질문은 다음과 같다.  
**기존 스마트폰에 이미 AI가 들어가 있는데, 왜 굳이 나만의 에이전트를 따로 만들어야 하는가?**

---

## 23장 요약. 스마트폰 내장 AI vs 나만의 에이전트

23장은 S26 같은 최신 스마트폰 내장 AI와 OpenClaw 기반 사용자 주도형 에이전트를 비교하면서, 단순한 기능 차이가 아니라 **구조와 철학의 차이**를 보여준다.

핵심 내용은 다음과 같다.
- 스마트폰 내장 AI는 잘 다듬어진 기능을 즉시 제공한다.
- 하지만 대부분 제공자가 정한 범위 안에서만 사용 가능하다.
- OpenClaw 기반 에이전트는 사용자가 직접 인터페이스, 모델, 도구, 스킬, 자동화, 문서 기억 구조를 설계할 수 있다.
- 내장 AI는 편리한 서비스에 가깝고, 나만의 에이전트는 설계 가능한 플랫폼에 가깝다.

이 장은 특히 세 가지 기준을 중심으로 비교를 정리한다.

### 1. S26 AI vs OpenClaw
- S26 AI: 제조사가 준비한 완성형 경험
- OpenClaw: 사용자가 조립하고 성장시킬 수 있는 실행형 플랫폼

### 2. 폐쇄형 vs 개방형
- 폐쇄형 AI는 사용하기 쉽지만 통제권이 제한된다.
- 개방형 에이전트는 설계 부담이 있지만 훨씬 높은 자유도와 실험 가능성을 제공한다.

### 3. 확장성 비교
- 내장 AI의 확장은 공급자 로드맵에 달려 있다.
- 나만의 에이전트는 사용자가 필요에 따라 문서, 툴, 자동화, 인터페이스를 계속 확장할 수 있다.

즉 이 장은 “누가 더 똑똑한가”보다, **누가 더 많이 통제하고 더 깊게 개인화하며 더 넓게 확장할 수 있는가**를 질문한다.

## PART 7의 핵심 정리

PART 7 전체를 관통하는 메시지는 분명하다.

1. 기존 스마트폰 내장 AI는 편리하지만, 대체로 제공된 범위 안에서 소비하는 구조다.  
2. 나만의 에이전트는 초기 설정 부담이 있지만, 그만큼 더 깊은 통제권과 확장성을 제공한다.  
3. 모바일 AI 에이전트의 진짜 가치는 단순한 기능이 아니라, **내 삶과 일에 맞춰 계속 자라날 수 있는 구조**에 있다.

즉 PART 7은 기존 AI를 부정하기 위한 장이 아니다.  
오히려 내장 AI가 강한 이유를 인정한 뒤에도, 왜 사용자 주도형 에이전트 플랫폼이 여전히 중요하고 가치 있는지를 설명하는 장이다.  
결국 이 파트는 “편리한 AI”와 “내가 키우는 AI 환경” 사이의 차이를 가장 분명하게 보여주는 비교 파트다.
