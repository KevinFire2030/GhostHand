# Remotion 소개 자료

## 어떤 프로젝트인지?

Remotion은 React로 영상을 만드는 프레임워크입니다.

일반적인 영상 편집 도구처럼 타임라인에서 컷을 직접 편집하는 방식이 아니라, React 컴포넌트와 TypeScript 코드로 각 프레임에 무엇을 보여줄지 정의합니다. 그 결과를 Remotion이 프레임 단위로 렌더링하고 MP4 같은 실제 영상 파일로 만들어 줍니다.

쉽게 말하면 다음과 같습니다.

```text
React 컴포넌트 + CSS 애니메이션 + 데이터
=> Remotion 렌더링
=> MP4 영상
```

React를 이미 알고 있다면 웹 화면을 만들듯이 영상 장면을 만들 수 있습니다. 텍스트, 이미지, SVG, Canvas, WebGL, Tailwind CSS 같은 웹 기술을 그대로 사용할 수 있고, 같은 템플릿에 다른 데이터를 넣어 여러 영상을 자동으로 만들 수도 있습니다.

## 어떻게 시작할 수 있는지

### 1. 준비물 확인

먼저 Node.js와 npm이 설치되어 있어야 합니다.

터미널에서 아래 명령을 실행해 봅니다.

```powershell
node --version
npm --version
```

버전이 출력되면 준비가 된 것입니다.

### 2. 새 Remotion 프로젝트 만들기

작업할 폴더로 이동합니다.

```powershell
cd E:\ax\PRJs\GhostHand\examples
```

새 프로젝트를 만듭니다.

```powershell
npx create-video@latest remotion-demo
```

템플릿을 고르라는 화면이 나오면 처음에는 `Hello World` 또는 `Blank`를 선택하면 됩니다.

프로젝트가 만들어지면 폴더로 이동합니다.

```powershell
cd remotion-demo
```

의존성을 설치합니다.

```powershell
npm i
```

### 3. Remotion Studio 실행하기

아래 명령으로 Remotion Studio를 실행합니다.

```powershell
npm run dev
```

브라우저에서 아래 주소를 엽니다.

```text
http://127.0.0.1:3000
```

여기서 영상을 미리 보고, 타임라인을 움직이며 특정 프레임을 확인할 수 있습니다.

## 간단한 데모 영상 만들기

아래는 10초짜리 간단한 소개 영상을 만드는 예시입니다.

목표:

```text
0~2초: 제목 등장
2~6초: 핵심 문구 표시
6~9초: 장점 3개 표시
9~10초: 마무리 문구
```

### 1. 컴포지션 길이 설정

`src/Root.tsx`를 엽니다.

30fps 기준으로 10초 영상은 300프레임입니다.

```tsx
<Composition
  id="DemoIntro"
  component={HelloWorld}
  durationInFrames={300}
  fps={30}
  width={1920}
  height={1080}
/>
```

중요한 값은 다음과 같습니다.

```text
durationInFrames: 전체 프레임 수
fps: 초당 프레임 수
width: 영상 너비
height: 영상 높이
```

### 2. 화면 컴포넌트 수정

`src/HelloWorld.tsx`를 열고 아래처럼 간단한 화면을 만들어 봅니다.

```tsx
import {
  AbsoluteFill,
  interpolate,
  useCurrentFrame,
} from "remotion";

export const HelloWorld: React.FC = () => {
  const frame = useCurrentFrame();

  const titleOpacity = interpolate(frame, [0, 30], [0, 1], {
    extrapolateRight: "clamp",
  });

  const subtitleOpacity = interpolate(frame, [60, 100], [0, 1], {
    extrapolateLeft: "clamp",
    extrapolateRight: "clamp",
  });

  const endingOpacity = interpolate(frame, [250, 290], [0, 1], {
    extrapolateLeft: "clamp",
    extrapolateRight: "clamp",
  });

  return (
    <AbsoluteFill
      style={{
        background: "#101820",
        color: "white",
        fontFamily: "Arial, sans-serif",
        justifyContent: "center",
        alignItems: "center",
      }}
    >
      <div style={{ fontSize: 96, fontWeight: 800, opacity: titleOpacity }}>
        GhostHand
      </div>

      <div style={{ fontSize: 44, marginTop: 32, opacity: subtitleOpacity }}>
        AI workflows from command to action
      </div>

      <div style={{ fontSize: 36, marginTop: 64, opacity: endingOpacity }}>
        Automate the next step.
      </div>
    </AbsoluteFill>
  );
};
```

여기서 핵심은 `useCurrentFrame()`입니다.

현재 프레임 번호를 가져온 다음, `interpolate()`로 프레임에 따라 투명도나 위치를 바꿉니다.

예를 들어:

```tsx
interpolate(frame, [0, 30], [0, 1])
```

이 코드는 0프레임에서 투명도 0, 30프레임에서 투명도 1이 되도록 값을 계산합니다. 즉 1초 동안 서서히 나타나는 애니메이션이 됩니다.

### 3. BGM 넣기

`public` 폴더에 오디오 파일을 넣습니다.

예:

```text
public/bgm.mp3
```

그다음 컴포넌트에서 불러옵니다.

```tsx
import { Audio, staticFile } from "remotion";

<Audio src={staticFile("bgm.mp3")} volume={0.3} />
```

`volume` 값은 0부터 1 사이로 조절합니다.

```text
0.1: 작게
0.3: 배경음악 정도
1.0: 원본 볼륨
```

### 4. MP4로 렌더링하기

미리보기에서 문제가 없으면 MP4로 렌더링합니다.

```powershell
npx remotion render DemoIntro out/demo-intro.mp4
```

렌더링이 끝나면 아래 위치에 영상 파일이 생깁니다.

```text
out/demo-intro.mp4
```

## 어디에 써먹을 수 있는지

Remotion은 한 번 만든 템플릿을 반복해서 재사용할 때 특히 좋습니다.

### 제품 소개 영상

서비스 이름, 기능, 가격, CTA 문구를 바꿔가며 여러 버전의 소개 영상을 만들 수 있습니다.

예:

```text
GhostHand 소개 영상
신규 기능 릴리즈 영상
랜딩 페이지용 짧은 데모 영상
```

### 쇼츠, 릴스, SNS 영상

세로 해상도인 `1080x1920`으로 설정하면 쇼츠나 릴스용 영상을 만들 수 있습니다.

```tsx
width={1080}
height={1920}
```

같은 내용을 가로형, 세로형, 정사각형으로 각각 렌더링할 수도 있습니다.

### 데이터 기반 리포트 영상

JSON이나 API 데이터를 받아서 차트, 수치, 순위 등을 영상으로 만들 수 있습니다.

예:

```text
주간 매출 리포트
사용자 증가 추이
GitHub 스타 증가 기록
서비스 운영 지표 요약
```

### 자동화된 마케팅 영상

고객 이름, 회사 이름, 추천 문구 등을 바꿔 개인화된 영상을 대량 생성할 수 있습니다.

예:

```text
고객별 환영 영상
세일즈 제안 영상
개인화된 사용 리포트 영상
```

### 개발 문서나 기능 소개

새 기능을 릴리즈할 때 짧은 설명 영상을 코드로 만들어 둘 수 있습니다.

코드로 관리되기 때문에 버전 관리가 쉽고, 나중에 문구나 색상만 바꿔 다시 렌더링하기 좋습니다.

## 정리

Remotion은 영상을 코드로 만드는 도구입니다.

처음에는 단순한 제목 애니메이션부터 시작하면 됩니다. 그다음 텍스트, 이미지, BGM, 데이터, 여러 장면을 조금씩 추가하면 제품 소개 영상이나 자동화 영상 템플릿으로 발전시킬 수 있습니다.

핵심 흐름은 아래와 같습니다.

```text
1. npx create-video@latest 로 프로젝트 생성
2. npm run dev 로 Studio 실행
3. React 컴포넌트로 장면 작성
4. npx remotion render 로 MP4 생성
```
