# 선행기술 검토의견

## 1. 검토 대상 기술
검토 대상은 동일 모바일 단말 내부에서 실행되는 인공지능 에이전트가 사용자의 자연어 명령을 해석하고, 동일 단말의 앱·운영체제 기능·시스템 설정을 직접 제어하며, 실행 결과를 검증하고 필요 시 대체 경로로 재시도하는 자기 단말 제어 기술이다.

본 기술은 단순한 음성명령 처리, 외부 PC 기반 단말 제어, 매크로 자동화, 접근성 기반 반복 제어와는 구별되며, 특히 다음 요소의 결합을 핵심 특징으로 한다.

- 동일 모바일 단말 내부에서 실행되는 AI 에이전트 구조
- 자연어 기반 자기 단말 제어
- 민감 작업에 대한 정책 판단 및 사용자 확인
- 복수 실행 경로 선택
- 실행 결과 검증 및 fallback 재시도
- 외부 PC 없이 단말 단독으로 동작 가능한 구조

## 2. 선행기술 조사 결과 요약
KIPRIS를 통해 1차 및 2차 검색을 수행한 결과, 일부 관련 가능성이 있는 문헌은 확인되었으나, 현재까지 확인된 범위에서는 본 발명의 핵심 구성을 동일하게 포함하는 선행기술은 명확히 발견되지 않았다.

특히 검색 결과 중 일부는 다음과 같은 점에서 부분 유사성이 있었다.

- 전자 장치에서의 제어 방법
- 지능형 에이전트 관리 구조
- 프롬프트 기반 사용자 인터페이스 제공 또는 제어
- 비음성 신호 기반 장치 제어

그러나 위 문헌들은 대체로 다음 한계를 가진다.

- 동일 단말 내부 AI 에이전트가 자기 단말을 직접 제어하는 구조가 명확하지 않음
- 자연어 해석부터 실제 단말 제어까지의 일체형 흐름이 불명확함
- 실행 결과 검증 및 실패 시 대체 경로 재실행 구조가 직접적으로 확인되지 않음
- 민감 작업 정책 판단 및 사용자 확인 경계가 충분히 구체화되어 있지 않음

## 3. 본 발명의 차별성
본 발명은 기존 선행기술 대비 다음과 같은 차별점을 가진다.

### (1) 동일 단말 내부 자기제어 구조
종래기술은 외부 호스트, 서버, 원격 제어 환경 또는 제한적 자동화 규칙에 의존하는 경우가 많다. 반면 본 발명은 AI 에이전트가 실행되는 단말과 실제 제어 대상 단말이 동일하다는 점에서 구조적 차별성이 있다.

### (2) 자연어 해석과 실제 제어의 직접 결합
일부 기술은 음성명령, UI 추천, 프롬프트 생성 등에 머무르지만, 본 발명은 사용자 자연어 명령을 실제 단말 제어 시퀀스로 직접 연결한다는 점에서 차이가 있다.

### (3) 검증 기반 폐루프 제어
본 발명은 단순 명령 수행에 그치지 않고, 실행 결과를 확인하고 목표 상태 미도달 시 다른 경로로 재시도하는 폐루프 구조를 가진다. 이는 신뢰성 측면에서 중요한 기술적 특징이다.

### (4) 정책 판단 및 민감 작업 보호
전화, 메시지, 파일 삭제, 금융 앱 접근, 권한 변경 등 민감 작업에 대해 정책 판단과 사용자 확인 절차를 반영할 수 있다는 점도 차별 요소로 볼 수 있다.

### (5) 복수 실행 경로 선택 구조
운영체제 API, 인텐트, 접근성 기반 UI 제어, 브리지 기반 명령 등 여러 실행 경로 중 상황에 맞는 경로를 선택하고, 실패 시 대체 경로로 전환하는 구조는 선행기술과 비교 시 유의미한 차별 포인트다.

## 4. 특허성 관점의 1차 의견
현재까지의 조사 결과를 기준으로 보면, 본 발명은 신규성 및 진보성을 주장할 여지가 있는 것으로 판단된다. 다만 이는 어디까지나 1차 선행기술 조사 결과 기준의 잠정 의견이며, 최종 판단을 위해서는 상위 유사 문헌의 초록, 청구항, 도면 및 상세설명에 대한 추가 검토가 필요하다.

특히 다음 포인트를 중심으로 권리화 전략을 정리하는 것이 바람직하다.

- “동일 단말 내부 AI 에이전트”라는 구조적 한정
- “자연어 해석 → 실행 경로 선택 → 제어 → 결과 검증 → fallback”의 폐루프 단계
- “민감 작업 정책 판단 및 사용자 확인” 요소
- “외부 PC 없이 단말 단독으로 동작”하는 점
- “복수 제어 경로의 선택 및 재선택” 구조

## 5. 검토의견 결론
종합하면, 현재 조사된 선행기술은 본 발명과 일부 개념적 접점은 있으나, 본 발명의 핵심 구성요소들이 결합된 형태를 직접적으로 개시한 문헌은 아직 확인되지 않았다. 따라서 본 발명은 자연어 기반 온디바이스 자기 단말 제어라는 점과 검증·재시도를 포함하는 폐루프 제어 구조를 중심으로 차별성을 주장할 수 있을 것으로 판단된다.

다만 향후 출원 또는 직무발명 제출 전에는 다음 보완이 필요하다.

- 상위 유사 문헌 3~5건에 대한 청구항 중심 정밀 비교
- 핵심 차별 포인트별 선행기술 맵 작성
- 독립항/종속항 관점의 권리범위 구체화
- 국내 외 해외 문헌까지 포함한 추가 조사
