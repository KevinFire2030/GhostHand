# 특허부서 전달사항

1. 본 발명은 동일 모바일 단말 내부에서 실행되는 인공지능 에이전트가 자연어 명령을 해석하여 자기 단말을 직접 제어하는 구조를 핵심으로 합니다.
2. 단순한 AI 명령 처리나 기존 모바일 자동화와의 차별점은, 자연어 해석 → 실행 경로 선택 → 실제 단말 제어 → 실행 결과 검증 → 실패 시 대체 경로 재시도로 이어지는 폐루프 제어 구조에 있습니다.
3. 또한 무선 디버깅 브리지 또는 로컬 브리지, 민감 작업에 대한 사용자 확인 경계, 복합 명령 분해 처리, 앱 빌드·설치·실행·제어 통합 가능성을 포함하는 점이 특징입니다.
4. 선행기술 조사 시에는 단순 모바일 자동화, 음성 비서, ADB 기반 제어뿐 아니라, 동일 단말 내부 AI 에이전트 기반 자기제어 구조와 결과 검증 및 fallback 구조를 포함하는 기술 여부를 중점적으로 검토할 필요가 있습니다.
5. 청구항 작성 시에는 동일 단말 내부 실행 구조, 자기 단말 식별, 복수 실행 경로 선택, 실행 결과 검증, 대체 경로 재선택, 민감 작업 확인을 중심으로 권리범위를 설계해 주시면 좋겠습니다.
6. 실시예 및 데모 참고자료
- 본 발명의 활용 예시, 데모 시나리오 및 시연 동영상을 직무발명 신고서와 함께 첨부하였으니, 발명의 이해를 위하여 참고 바랍니다.
