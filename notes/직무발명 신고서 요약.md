# 직무발명 신고서 요약

## 1. 발명의 명칭
**동일 모바일 단말 내 인공지능 에이전트를 이용한 자연어 기반 자기 단말 제어 장치 및 방법**

## 2. 발명의 개요
본 발명은 동일 모바일 단말 내부에서 실행되는 인공지능 에이전트가 사용자의 자연어 명령을 해석하고, 해석된 결과에 따라 자기 단말의 앱, 운영체제 기능 및 시스템 설정을 직접 제어하는 기술에 관한 것이다. 본 발명은 자연어 해석, 자기 단말 식별, 민감 작업 정책 판단, 실행 경로 선택, 실제 단말 제어, 실행 결과 검증 및 실패 시 대체 경로 재선택을 포함하는 폐루프 구조를 통해, 외부 개인용 컴퓨터 없이도 단말 단독으로 동작 가능한 자기제어 아키텍처를 제공한다.

## 3. 발명의 핵심 포인트
- 동일 모바일 단말 내부에서 실행되는 AI 에이전트 구조
- 동일 단말에 대한 자연어 기반 자기제어 구조
- 실행 결과 검증 및 대체 경로 재시도를 포함하는 폐루프 제어
- 무선 디버깅 브리지 또는 로컬 브리지 기반 제어 경로
- 민감 작업에 대한 사용자 확인 경계 설계
- 앱 빌드·설치·실행·제어 통합 가능 구조

## 4. 종래기술 대비 비교표

| 구분 | 종래기술 | 본발명 |
| --- | ---- | --- |
| 목적 | 외부 PC, 서버 또는 단순 자동화 도구를 이용하여 모바일 단말의 일부 기능을 원격 또는 반복 제어 | 동일 모바일 단말 내부에서 실행되는 AI 에이전트가 자연어 명령을 해석하여 자기 단말을 직접 제어 |
| 구성 | 외부 호스트, 디버깅 인터페이스, 스크립트, 매크로, 접근성 기반 자동화 또는 단순 음성 비서 중심 | 입력부, AI 에이전트 실행부, 자연어 해석부, 자기 단말 인식부, 정책 판단부, 실행 경로 선택부, 무선 디버깅/로컬 브리지부, 제어 실행부, 실행 결과 검증부, 대체 경로 선택부, 권한 및 보안 관리부 |
| 효과 | 일부 반복 작업 자동화, 제한적 원격 제어, 단순 편의 기능 제공 | 자연어 기반 직관적 단말 제어, 결과 검증 기반 신뢰성 향상, 민감 작업 보호, 외부 PC 없는 단말 단독 제어 가능 |
| 차이점 | 자연어 해석과 실제 제어가 분리되거나, 외부 장치 의존성이 높고, 실패 시 재시도 구조가 미흡함 | 동일 단말 내부 AI 에이전트가 자연어 해석부터 실행·검증·fallback까지 수행하는 자기제어 폐루프 구조를 가짐 |

## 5. 기대 효과
1. 사용자는 복잡한 스크립트나 메뉴 탐색 없이 자연어로 단말을 제어할 수 있다.
2. 실행 결과 검증 및 재시도 구조를 통해 제어 신뢰성이 향상된다.
3. 민감 작업에 대한 사용자 확인을 통해 보안성과 안전성이 향상된다.
4. 모바일 자동화, 온디바이스 AI 서비스, 접근성 보조 및 개발자 모드 기반 자동화 분야에 폭넓게 적용 가능하다.

## 6. 적용 가능성
- **제품/과제 적용 발명 여부:** 적용 가능
- **제품/과제명:** 온디바이스 AI 에이전트 기반 모바일 단말 제어 기술 개발 과제
- **기능/서비스명:** 자연어 기반 자기 단말 제어 서비스, 민감 작업 확인 기반 단말 자동화 기능
- **적용(예정) 시점:** 2026년 파일럿 검토 및 선행개발 과제 적용 예정
