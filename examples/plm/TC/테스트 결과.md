# 테스트 결과

## 개요
PLM 기본 검증 시나리오 5건을 수행하고 실행 결과를 정리하였다.

| TC | 항목 | 예상 결과 | 실제 결과 | 판정 | 비고 |
|---|---|---|---|---|---|
| TC1 | 전화 걸기 | 심카드 미삽입 상태로 정상 발신 불가 | 전화 앱/다이얼 화면 진입 확인, 정상 발신 불가 환경 | FAIL | 문제점 등록 대상 |
| TC2 | 문자 보내기 | 심카드 미삽입 상태로 정상 발신 불가 | 메시지 작성 화면 진입 확인, 정상 발신 불가 환경 | FAIL | 문제점 등록 대상 |
| TC3 | 홈화면 이동 | 홈화면으로 즉시 이동 | 홈화면 이동 확인 | PASS |  |
| TC4 | 유튜브 앱 실행 | 유튜브 앱 실행 | 유튜브 앱 실행 확인 | PASS |  |
| TC5 | 알람설정 | 지정 시간 알람 설정 가능 | 알람 설정/알람 목록 진입 확인 | PASS |  |
