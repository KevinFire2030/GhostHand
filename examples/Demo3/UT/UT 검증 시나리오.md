# UT 검증 시나리오

목적: 기본 스마트폰 사용 시나리오를 기준으로 온디바이스 AI Agent가 단말 기능을 검증하고, 실행 결과를 캡처/녹화/로그와 함께 정리할 수 있도록 한다.

평가 기준:
- PASS = 기대 동작이 정상 수행됨
- FAIL = 동작 실패, 환경 문제, 앱/시스템 오류, 자동화 실패 포함

---

## TC-01 전화 걸기 - 홈으로 이동
**목적**
- 전화 앱 실행 및 발신 시도 동작이 정상인지 확인

**절차**
1. 전화 앱을 실행한다.
2. 지정한 번호 또는 테스트 번호 입력 화면으로 진입한다.
3. 전화 걸기를 시도한다.
4. 결과를 확인한 뒤 홈 화면으로 이동한다.

**확인 항목**
- 전화 앱이 정상 실행되는가
- 다이얼 입력 및 발신 버튼 동작이 가능한가
- SIM 미삽입/통신 불가 시 적절한 안내가 표시되는가
- 종료 후 홈으로 정상 복귀하는가

**합격 기준**
- 전화 앱 실행 및 발신 시도 흐름이 정상 수행되고, 실패 시에도 의도한 안내가 확인된다.

**확인 항목**

- 검색어 입력이 정상 동작하는가
- 검색 결과가 정상 표시되는가
- 종료 후 홈으로 정상 복귀하는가

**합격 기준**

## TC-05 시계 앱 실행 - 07:00 알람 설정 - 알람 해제 - 홈으로 이동
**목적**
- 시계 앱 내 알람 생성 및 해제 기능이 정상 동작하는지 확인

**절차**
1. 시계 앱을 실행한다.
2. 알람 메뉴로 이동한다.

6. 홈 화면으로 이동한다.

**확인 항목**
- 시계 앱이 정상 실행되는가
- 알람 추가가 가능한가
- 07:00 설정이 정확히 저장되는가
- 알람 해제가 정상 동작하는가
- 종료 후 홈으로 정상 복귀하는가

**합격 기준**
- 시계 앱 실행부터 알람 설정/해제, 홈 복귀까지 정상 수행된다.
