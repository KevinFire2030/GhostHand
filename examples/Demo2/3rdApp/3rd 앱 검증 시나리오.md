# 3rd 앱 검증 시나리오

## 1. 개요
본 문서는 `examples/Demo2/3rdApp` 경로 기준의 3rd 앱 검증 시나리오를 정의한다.
테스트 케이스(TC)는 총 10개로 구성한다.
일반 사용자가 일상에서 자주 사용하는 기능 위주로 검증할 수 있는 케이스를 기준으로 작성한다.

## 2. 테스트 케이스

### TC1. 구글 플레이 앱 실행
- 목적: 구글 플레이 앱이 정상 실행되는지 확인한다.
- 절차:
  1. 구글 플레이 앱을 실행한다.
  2. 메인 화면이 표시되는지 확인한다.
- 예상 결과:
  - 구글 플레이 앱이 비정상 종료 없이 정상 실행된다.
- 예상 판정: PASS

### TC2. 인스타그램 검색
- 목적: 구글 플레이에서 인스타그램 검색이 가능한지 확인한다.
- 절차:
  1. 검색창을 선택한다.
  2. `Instagram` 또는 `인스타그램`을 입력한다.
  3. 검색 결과를 확인한다.
- 예상 결과:
  - 인스타그램 검색 결과가 정상 표시된다.
- 예상 판정: PASS

### TC3. 인스타그램 앱 상세 진입
- 목적: 인스타그램 앱 상세 페이지 진입 여부를 확인한다.
- 절차:
  1. 검색 결과에서 인스타그램 앱을 선택한다.
  2. 상세 페이지로 진입한다.
- 예상 결과:
  - 인스타그램 앱 상세 정보가 정상 표시된다.
- 예상 판정: PASS

### TC4. 인스타그램 다운로드 및 설치
- 목적: 인스타그램 앱 다운로드 및 설치 동작을 확인한다.
- 절차:
  1. 설치 버튼을 선택한다.
  2. 다운로드 및 설치 진행 상태를 확인한다.
- 예상 결과:
  - 다운로드 및 설치가 정상 완료된다.
- 예상 판정: PASS

### TC5. 인스타그램 앱 실행
- 목적: 설치 완료 후 인스타그램 앱이 정상 실행되는지 확인한다.
- 절차:
  1. 열기 버튼 또는 앱 아이콘을 선택한다.
  2. 초기 실행 화면을 확인한다.
- 예상 결과:
  - 인스타그램 앱이 정상 실행된다.
- 예상 판정: PASS

### TC6. 로그인 화면 표시 확인
- 목적: 인스타그램 로그인 화면이 정상 표시되는지 확인한다.
- 절차:
  1. 앱 실행 후 로그인 화면으로 진입한다.
  2. 아이디/비밀번호 입력창과 로그인 버튼을 확인한다.
- 예상 결과:
  - 로그인 화면 구성 요소가 정상 표시된다.
- 예상 판정: PASS

### TC7. 회원가입 또는 계정 찾기 링크 확인
- 목적: 로그인 화면 내 주요 링크 동작 가능 여부를 확인한다.
- 절차:
  1. 회원가입 또는 계정 찾기 링크를 확인한다.
  2. 선택 후 화면 전환 여부를 확인한다.
- 예상 결과:
  - 링크 선택 시 해당 화면으로 정상 이동한다.
- 예상 판정: PASS

### TC8. 하단 네비게이션 또는 초기 UI 확인
- 목적: 앱 실행 후 기본 UI 구성이 정상인지 확인한다.
- 절차:
  1. 앱 초기 화면 또는 로그인 화면의 주요 UI를 확인한다.
  2. 버튼, 아이콘, 로고, 링크 배치 상태를 확인한다.
- 예상 결과:
  - UI가 깨짐 없이 정상 표시된다.
- 예상 판정: PASS

### TC9. 앱 재실행 안정성 확인
- 목적: 앱 종료 후 재실행 시 정상 동작하는지 확인한다.
- 절차:
  1. 인스타그램 앱을 종료한다.
  2. 다시 실행한다.
  3. 초기 화면 표시 여부를 확인한다.
- 예상 결과:
  - 앱이 재실행 후에도 정상 동작한다.
- 예상 판정: PASS

### TC10. 홈 화면 복귀 및 앱 아이콘 확인
- 목적: 설치 후 홈 화면에서 앱 아이콘이 정상 표시되는지 확인한다.
- 절차:
  1. 홈 화면으로 이동한다.
  2. 인스타그램 앱 아이콘 표시 여부를 확인한다.
- 예상 결과:
  - 홈 화면 또는 앱 서랍에서 인스타그램 아이콘이 정상 표시된다.
- 예상 판정: PASS
