# 테마앱 검증 시나리오

## 1. 개요
본 문서는 `examples/Demo2/ThemeApp` 경로 기준의 테마앱 검증 시나리오를 정의한다.
테스트 케이스(TC)는 총 10개로 구성한다.
무료 테마 검색, 다운로드, 설치, 적용 후 주요 UI/화면 표시 이상 유무를 검증 대상으로 한다.

## 2. 테스트 케이스

### TC1. 테마 앱 실행
- 목적: 테마 앱이 정상 실행되는지 확인한다.
- 절차:
  1. 테마 앱을 실행한다.
  2. 메인 화면이 표시되는지 확인한다.
- 예상 결과:
  - 테마 앱이 비정상 종료 없이 정상 실행된다.
- 예상 판정: PASS

### TC2. 무료 테마 검색
- 목적: 무료 테마 목록 및 검색 기능을 확인한다.
- 절차:
  1. 무료 테마 메뉴 또는 탭으로 이동한다.
  2. 무료 테마 목록이 노출되는지 확인한다.
- 예상 결과:
  - 무료 테마 목록이 정상 표시된다.
- 예상 판정: PASS

### TC3. 무료 테마 상세 진입
- 목적: 무료 테마 상세 페이지 진입 여부를 확인한다.
- 절차:
  1. 무료 테마 하나를 선택한다.
  2. 상세 페이지로 진입한다.
- 예상 결과:
  - 선택한 테마의 상세 정보가 정상 표시된다.
- 예상 판정: PASS

### TC4. 무료 테마 다운로드
- 목적: 무료 테마 다운로드 동작을 확인한다.
- 절차:
  1. 상세 페이지에서 다운로드 버튼을 선택한다.
  2. 다운로드 진행 상태를 확인한다.
- 예상 결과:
  - 다운로드가 정상 시작되고 완료된다.
- 예상 판정: PASS

### TC5. 무료 테마 설치 및 적용
- 목적: 무료 테마 설치 및 적용 동작을 확인한다.
- 절차:
  1. 다운로드 완료 후 설치 또는 적용을 선택한다.
  2. 테마 적용 완료 여부를 확인한다.
- 예상 결과:
  - 선택한 테마가 정상 적용된다.
- 예상 판정: PASS

### TC6. 홈 화면 UI 확인
- 목적: 테마 적용 후 홈 화면 UI 이상 유무를 확인한다.
- 절차:
  1. 홈 화면으로 이동한다.
  2. 아이콘, 배경, 위젯, 배치 상태를 확인한다.
- 예상 결과:
  - 홈 화면 UI가 정상 표시된다.
- 예상 판정: PASS

### TC7. 잠금 화면 UI 확인
- 목적: 테마 적용 후 잠금 화면 UI 이상 유무를 확인한다.
- 절차:
  1. 화면을 잠근다.
  2. 잠금 화면의 배경 및 시계/텍스트 표시를 확인한다.
- 예상 결과:
  - 잠금 화면 UI가 정상 표시된다.
- 예상 판정: PASS

### TC8. 글자 깨짐 여부 확인
- 목적: 테마 적용 후 한글/영문 텍스트 깨짐 여부를 확인한다.
- 절차:
  1. 홈 화면 및 테마 상세 화면의 텍스트를 확인한다.
  2. 한글/영문 글자가 비정상 표시되는지 확인한다.
- 예상 결과:
  - 글자 깨짐 없이 정상 표시된다.
- 예상 판정: PASS

### TC9. 아이콘/리소스 깨짐 여부 확인
- 목적: 테마 적용 후 아이콘 및 이미지 리소스 깨짐 여부를 확인한다.
- 절차:
  1. 홈 화면 및 앱 목록 화면을 확인한다.
  2. 아이콘 또는 그래픽이 비정상 표시되는지 확인한다.
- 예상 결과:
  - 아이콘 및 리소스가 정상 표시된다.
- 예상 판정: PASS

### TC10. 기본 테마 복귀
- 목적: 적용한 테마를 기본 테마로 복귀할 수 있는지 확인한다.
- 절차:
  1. 테마 앱 또는 설정으로 이동한다.
  2. 기본 테마를 선택한다.
  3. 기본 테마 적용 완료 여부를 확인한다.
- 예상 결과:
  - 기본 테마로 정상 복귀된다.
- 예상 판정: PASS
