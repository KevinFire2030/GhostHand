# HeyGen으로 아바타 영상 직접 만들어보기

오늘은 HeyGen API를 활용해서 간단한 아바타 영상을 직접 생성해본 과정을 정리해보려고 합니다.  
결론부터 말하면, **API Key 발급 → 아바타/보이스 확인 → Python 코드 실행 → mp4 다운로드** 흐름으로 비교적 빠르게 테스트할 수 있었습니다.

---

## 1. 왜 해봤나?
요즘 AI 기반 아바타 영상 도구들이 많지만, 실제로 API까지 붙여서 자동화할 수 있는지가 더 중요하다고 생각했습니다.  
단순히 웹에서 버튼을 눌러 영상을 만드는 것이 아니라, 코드로 호출해서 자동 생성하고 결과 파일까지 저장할 수 있어야 활용도가 높기 때문입니다.

특히 발표 영상, 설명 영상, 브리핑 영상처럼 반복적으로 만들어야 하는 콘텐츠는 API 자동화와 잘 맞습니다.

---

## 2. 준비한 것
이번 테스트에서 준비한 것은 크게 3가지였습니다.

- HeyGen API Key
- 사용할 avatar_id
- 사용할 voice_id

그리고 프로젝트 폴더 아래에 `.env` 파일을 만들어서 아래처럼 관리했습니다.

```env
HEYGEN_API_KEY=...
HEYGEN_AVATAR_ID=...
HEYGEN_VOICE_ID=...
HEYGEN_INPUT_TEXT=안녕하세요. HeyGen API를 활용한 아바타 영상 생성 테스트입니다.
```

이렇게 해두면 코드 안에 민감한 값들을 직접 넣지 않아도 되고, 나중에 값만 바꿔서 재사용하기도 편합니다.

---

## 3. 테스트 과정
전체 흐름은 생각보다 단순했습니다.

### (1) API 연결 확인
먼저 API Key가 정상인지 확인하기 위해 HeyGen API에 간단한 조회 요청을 보냈습니다.  
이 단계에서 응답이 200으로 오면 인증은 정상입니다.

### (2) voice_id 확인
다음으로 음성 목록 조회 API를 호출해서 사용할 수 있는 voice_id를 확인했습니다.  
조회 결과를 보니 기본 제공 음성뿐 아니라 제가 만든 한국어 음성도 함께 확인할 수 있었습니다.

### (3) avatar_id 설정
아바타는 미리 확보한 avatar_id를 사용했습니다.  
기본 제공 아바타가 아니라 개인화된 아바타를 활용해 테스트했습니다.

### (4) 영상 생성 Python 코드 작성
이후 Python으로 영상 생성 코드를 작성했습니다.  
이 코드는 `.env`에서 API Key, avatar_id, voice_id, input_text를 읽어와 HeyGen API를 호출합니다.

핵심 흐름은 다음과 같습니다.

1. `.env` 로드
2. 영상 생성 요청
3. `video_id` 받기
4. 상태 조회 polling
5. 완료되면 mp4 다운로드
6. 결과 json/txt 저장

---

## 4. 중간에 겪은 문제
처음부터 한 번에 된 것은 아니었습니다.

### 크레딧 부족 에러
초기 테스트에서는 영상 생성 요청 자체는 성공했지만, 실제 렌더링 단계에서 아래와 같은 에러가 발생했습니다.

- `MOVIO_PAYMENT_INSUFFICIENT_CREDIT`

즉, API 인증은 정상인데 **API 크레딧이 부족해서 렌더링이 실패한 상황**이었습니다.

이 문제는 HeyGen 대시보드에서 API 크레딧을 확인하고 다시 시도하면서 해결했습니다.

### `.env` 인코딩 문제
또 하나는 `.env` 파일 앞에 BOM이 들어가 있어서 API Key를 제대로 못 읽는 문제가 있었습니다.  
이 부분은 UTF-8 인코딩을 정리한 뒤 해결했습니다.

이런 사소한 인코딩 문제도 자동화에서는 꽤 중요하다는 걸 다시 느꼈습니다.

---

## 5. 최종 결과
최종적으로는 영상 생성부터 다운로드까지 정상 동작했습니다.

자동화 결과물은 아래처럼 저장되도록 구성했습니다.

- `output/<video_id>.mp4` : 실제 영상 파일
- `output/<video_id>.json` : API 응답 원문
- `output/<video_id>.txt` : video_url, thumbnail_url 등 핵심 정보

즉, 단순히 영상을 생성하는 데서 끝나는 것이 아니라, **결과 파일 관리까지 한 번에 되도록 자동화**한 것입니다.

---

## 6. 느낀 점
HeyGen은 생각보다 API 사용 흐름이 명확했습니다.  
한 번 세팅해두면 발표 영상, 소개 영상, 브리핑 영상 같은 반복 작업에 바로 적용할 수 있을 것 같았습니다.

특히 좋았던 점은 아래와 같습니다.

- API 기반 자동화가 가능하다
- 아바타/보이스를 조합해서 빠르게 테스트할 수 있다
- 결과 파일(mp4/json/txt)까지 함께 관리할 수 있다

반면 주의할 점도 있었습니다.

- API 크레딧과 웹 사용 크레딧은 구분해서 봐야 한다
- `.env` 인코딩이나 키 로딩 문제를 조심해야 한다
- 아바타 타입에 따라 비용이 꽤 달라질 수 있다

---

## 7. 마무리
오늘 테스트를 통해 HeyGen은 단순한 데모용 툴이 아니라, **실제로 자동화 파이프라인에 연결할 수 있는 영상 생성 API**라는 점을 확인했습니다.

앞으로는 여기에 스크립트 파일 분리, 여러 문장 일괄 생성, 음성 기반 입력 처리까지 붙이면 더 실용적인 워크플로우를 만들 수 있을 것 같습니다.

간단히 정리하면,
**HeyGen은 “아바타 영상을 손으로 만드는 도구”를 넘어서, 코드로 연결해 자동 생성할 수 있는 실무형 영상 API”**라고 볼 수 있겠습니다.
