# 시나리오_v1

## 1. 직무발명 1장 요약 (1분)
- 동일 모바일 단말 내부에서 실행되는 AI 에이전트가 동일 단말을 자연어로 직접 제어하는 구조
- 외부 PC/USB 연결 중심의 기존 자동화 구조를 단말 내부 자기제어 구조로 단순화
- 자연어 해석 → 실행 경로 선택 → 결과 검증 → 재시도까지 포함하는 폐루프 자동화 방식

---

## 2. 활용예시 (2분)
### Theme App 검증 자동화

### As-is
- 테스트 단말
  - 테마 앱 다운로드 및 설치
- 자동화 서버
  - USB로 테스트 단말 연결
  - 테스트 스크립트 실행
  - 테스트 결과 이미지로 저장 (100장)
  - AWS S3에 업로드
- Azure 클라우드
  - 테마앱 검증 에이전트 실행
  - S3에서 이미지 다운로드
  - 판독 및 판정 (Pass, Fail, Reason)
- WIND 시스템에 결과 업로드

### To-be
- 테스트 단말
  - 사용자 자연어로 테마앱 검증 요청
- 오픈클루
  - 검증 시나리오 작성
  - 테마앱 다운로드/설치/실행
  - 결과 저장
  - 판정
  - 결과 WIND 시스템에 업로드

## 3. 기대효과 (1분)
### 테스트 간소화
- 테스트 단말에서 대부분의 절차를 직접 수행
- 절차 단축 및 작업 흐름 단순화

### 비용 절감
- 구축 비용 절감
- 운영 비용 절감

### 운영 유연성
- 단말이 어디에 있든 인터넷 연결만 되어 있으면 제어 가능
- 글로벌 운영 환경 대응 가능
  - 6개국 32명 대상 운영/확장 가능성
